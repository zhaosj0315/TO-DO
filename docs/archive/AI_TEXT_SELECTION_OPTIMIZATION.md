# 🎯 文本选择AI功能优化方案

**创建日期**: 2026-03-02  
**问题类型**: 用户体验优化  
**优先级**: P1（高优先级）

---

## 🐛 当前问题

### **问题描述**
用户在页面任何地方选中文本都会弹出10个AI功能菜单，包括：
1. ✨ 改进
2. 📏 精简
3. 📐 扩展
4. 🔄 语气
5. 🌐 翻译
6. 📝 修正
7. 💡 解释
8. 🎯 要点
9. 🤖 续写
10. 📋 提取任务

### **用户痛点**
1. **误触发频繁**: 用户复制文本、选中查看时都会弹出菜单
2. **干扰阅读**: 选中文本查看时，菜单遮挡内容
3. **不够智能**: 没有根据上下文判断是否需要AI功能
4. **范围过广**: 页面所有文本都可触发，包括标题、按钮文字等

---

## 🎯 优化目标

### **核心原则**
1. **精准触发**: 只在用户真正需要AI处理的场景下显示
2. **减少干扰**: 不影响正常的文本选择和复制操作
3. **智能判断**: 根据选中文本的位置和内容智能决定是否显示
4. **用户可控**: 提供开关选项，让用户自主决定

---

## 💡 优化方案

### **方案1: 限制触发区域（推荐）** ⭐⭐⭐⭐⭐

#### 实现逻辑
只在以下区域触发AI文本菜单：
1. ✅ **任务描述输入框**（全屏编辑模式）
2. ✅ **任务详情页的描述区域**
3. ✅ **日志内容输入框**
4. ✅ **日志详情的内容区域**
5. ❌ **任务标题**（不触发）
6. ❌ **按钮文字**（不触发）
7. ❌ **导航栏**（不触发）
8. ❌ **统计数字**（不触发）

#### 优点
- 精准定位用户需求场景
- 大幅减少误触发
- 不影响正常文本选择

#### 缺点
- 需要明确定义触发区域

#### 实现难度
- 🟢 简单：修改`useTextSelection.js`的容器检测逻辑

---

### **方案2: 增加最小字数限制** ⭐⭐⭐⭐

#### 实现逻辑
只有选中文本达到一定字数才触发菜单：
- 最小字数：10个字符（中文5个字）
- 最大字数：5000个字符

#### 优点
- 过滤掉单词、短语的误触发
- 实现简单

#### 缺点
- 无法完全解决问题
- 用户可能需要处理短文本

#### 实现难度
- 🟢 简单：添加字数判断

---

### **方案3: 双击或长按触发** ⭐⭐⭐

#### 实现逻辑
- 单次选中：不触发菜单
- 双击选中：触发菜单
- 长按选中（移动端）：触发菜单

#### 优点
- 明确的用户意图
- 完全避免误触发

#### 缺点
- 改变用户习惯
- 增加操作步骤

#### 实现难度
- 🟡 中等：需要区分单击和双击

---

### **方案4: 添加快捷键触发** ⭐⭐⭐⭐

#### 实现逻辑
- 选中文本后，按`Ctrl/Cmd + K`触发菜单
- 或者在选中文本后显示一个小图标，点击图标触发菜单

#### 优点
- 完全由用户控制
- 不干扰正常操作

#### 缺点
- 需要用户学习快捷键
- 移动端不适用

#### 实现难度
- 🟡 中等：需要添加快捷键监听

---

### **方案5: 智能上下文判断（最智能）** ⭐⭐⭐⭐⭐

#### 实现逻辑
根据选中文本的特征智能判断是否显示菜单：

**触发条件（满足任一即可）**：
1. 选中文本在可编辑区域（textarea, contenteditable）
2. 选中文本长度 > 10个字符
3. 选中文本包含标点符号（说明是句子）
4. 选中文本在特定容器内（任务描述、日志内容）

**不触发条件（优先级更高）**：
1. 选中文本是纯数字
2. 选中文本是单个单词（无空格、无标点）
3. 选中文本在按钮、链接内
4. 选中文本在导航栏、标题栏

#### 优点
- 最智能的解决方案
- 用户体验最好
- 几乎无误触发

#### 缺点
- 实现复杂度较高
- 需要大量测试

#### 实现难度
- 🔴 复杂：需要综合判断多个条件

---

## 🎯 推荐方案

### **组合方案：方案1 + 方案2 + 方案5部分逻辑**

#### 实现步骤

**第1步：限制触发区域（方案1）**
```javascript
// useTextSelection.js
const handleMouseUp = (event) => {
  setTimeout(() => {
    const selection = window.getSelection()
    const text = selection.toString().trim()

    if (!text) {
      showMenu.value = false
      return
    }

    // ✅ 检查是否在允许的容器内
    const allowedContainers = [
      '.fullscreen-desc-textarea',  // 全屏编辑
      '.task-description',           // 任务描述
      '.log-content',                // 日志内容
      '.editable-content'            // 可编辑内容
    ]
    
    let isInAllowedContainer = false
    for (const selector of allowedContainers) {
      const container = document.querySelector(selector)
      if (container && container.contains(selection.anchorNode)) {
        isInAllowedContainer = true
        break
      }
    }
    
    if (!isInAllowedContainer) {
      console.log('useTextSelection - not in allowed container')
      return
    }

    // 继续原有逻辑...
  }, 10)
}
```

**第2步：增加最小字数限制（方案2）**
```javascript
// 最小字数限制
const MIN_TEXT_LENGTH = 10

if (text.length < MIN_TEXT_LENGTH) {
  console.log('useTextSelection - text too short')
  return
}
```

**第3步：过滤特殊情况（方案5部分）**
```javascript
// 过滤纯数字
if (/^\d+$/.test(text)) {
  console.log('useTextSelection - pure number')
  return
}

// 过滤单个单词（无空格、无标点）
if (!/[\s\p{P}]/u.test(text) && text.length < 20) {
  console.log('useTextSelection - single word')
  return
}

// 检查是否在按钮、链接内
const element = selection.anchorNode.parentElement
if (element.closest('button') || element.closest('a')) {
  console.log('useTextSelection - in button or link')
  return
}
```
