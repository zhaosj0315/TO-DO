## 🎯 触发条件（5个过滤规则）

### **1. 最小字数限制** ✅
- **规则**: 选中文本 ≥ 10个字符
- **目的**: 过滤单词、短语的误触发

### **2. 过滤纯数字** ✅
- **规则**: 纯数字不触发（如：12345）
- **目的**: 避免选中统计数字时弹出菜单

### **3. 过滤单个单词** ✅
- **规则**: 无空格、无标点且长度<20的文本不触发
- **目的**: 避免选中单个单词时弹出菜单

### **4. 限制触发区域** ✅
- **规则**: 只在以下元素内触发
  - 所有 `<textarea>` 元素
  - 所有 `contenteditable="true"` 的元素
  - 所有包含 `.editable-content` 类的元素
  - 包含textarea的父元素（如section）
- **目的**: 避免在标题、按钮、导航栏等区域触发

### **5. 过滤按钮和链接** ✅
- **规则**: 按钮和链接内的文字不触发
- **目的**: 避免选中按钮文字时弹出菜单

### **短文本**
- ❌ 少于10个字符的文本
- ❌ 纯数字
- ❌ 单个单词（无空格、无标点）

## 🔧 技术实现

### **核心文件**
- `src/composables/useTextSelection.js` - 文本选择逻辑
- `src/components/AITextMenu.vue` - AI菜单组件
- `src/services/aiTextService.js` - AI文本处理服务

### **触发逻辑**
```javascript
// 1. 监听mouseup事件
// 2. 获取选中文本
// 3. 检查5个过滤条件
// 4. 计算菜单位置
// 5. 显示AI菜单
```

### **过滤条件代码**
```javascript
// 1. 最小字数
if (text.length < 10) return

// 2. 过滤纯数字
if (/^\d+$/.test(text)) return

// 3. 过滤单词
if (!/[\s\p{P}]/u.test(text) && text.length < 20) return

// 4. 限制区域
if (!isInTextarea) return

// 5. 过滤按钮
if (element.closest('button')) return
```
