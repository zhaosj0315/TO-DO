## 📝 使用说明

### 快速开始

1. **创建任务A**：
   ```
   标题：项目规划
   描述：制定整体规划
   ```

2. **创建任务B并引用A**：
   ```
   标题：需求文档
   描述：参考 [[项目规划]] 编写需求文档 #work/project-a
   ```

3. **查看效果**：
   - 任务B描述中 [[项目规划]] 显示为紫色链接
   - 点击链接跳转到任务A
   - 任务A详情页底部显示反向链接：需求文档
   - 标签 #work/project-a 显示为紫色徽章
