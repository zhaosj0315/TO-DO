## 📋 实施内容

### 核心功能

#### 1. 标签自动补全 🏷️

**触发**：输入 `#`

**功能**：
- 自动提示已有标签
- 支持模糊搜索
- 显示标签使用次数
- 键盘导航（↑↓ Enter Esc）

**示例**：
```
输入：#wo
提示：
  🏷️ #work (5)
  🏷️ #work/project-a (3)
  🏷️ #work/project-b (2)
```

#### 2. 任务链接自动补全 🔗

**触发**：输入 `[[`

**功能**：
- 自动提示任务名称
- 支持模糊搜索
- 显示任务优先级（颜色图标）
- 键盘导航（↑↓ Enter Esc）

**示例**：
```
输入：[[项目
提示：
  🔴 项目规划
  🟡 项目A需求
  🔵 项目B设计
```

## 🔧 技术实现

### 1. AutocompleteDropdown 组件

```vue
<!-- src/components/AutocompleteDropdown.vue -->
- 下拉列表展示
- 键盘导航支持
- 鼠标悬停高亮
- 点击选择
```

**特点**：
- 固定定位（跟随光标）
- 最多显示10条建议
- 自动滚动
- Esc 关闭

### 2. useAutocomplete Composable

```javascript
// src/composables/useAutocomplete.js
- 触发器检测（# 和 [[）
- 建议生成（标签和任务）
- 位置计算（光标位置）
- 文本替换（选择后）
```

**算法**：
1. 监听 input 事件
2. 获取光标前的文本
3. 正则匹配触发器
4. 生成建议列表
5. 计算下拉位置
6. 显示下拉框

### 3. 集成到 TodoView

```vue
<!-- 任务描述输入框 -->
<textarea
  ref="descriptionTextarea"
  @input="handleAutocompleteInput"
  placeholder="提示：输入 # 添加标签，输入 [[ 链接任务"
/>

<!-- 自动补全下拉 -->
<AutocompleteDropdown
  :show="showAutocomplete"
  :suggestions="autocompleteSuggestions"
  @select="selectAutocompleteSuggestion"
/>
```

```
4. 结果："参考 [[项目规划]] "

```
