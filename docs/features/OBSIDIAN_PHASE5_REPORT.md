## 🔧 技术细节

### ECharts 配置

```javascript
{
  type: 'graph',
  layout: 'force',
  roam: true,  // 支持缩放和拖拽
  force: {
    repulsion: 200,    // 节点斥力
    edgeLength: 150,   // 边长度
    gravity: 0.1       // 重力
  },
  emphasis: {
    focus: 'adjacency'  // 高亮相邻节点
  }
}
```

### 关系深度算法

```javascript
function getRelatedTasks(taskId) {
  const related = new Set()
  const queue = [taskId]
  const visited = new Set()
  let depth = 0

  while (queue.length > 0 && depth < 2) {
    // BFS 遍历2层
    const levelSize = queue.length
    for (let i = 0; i < levelSize; i++) {
      const currentId = queue.shift()
      if (visited.has(currentId)) continue
      visited.add(currentId)

      const task = taskStore.tasks.find(t => t.id === currentId)
      if (task) {
        related.add(task)
        // 添加所有关联任务到队列
        task.linkedTasks?.forEach(id => queue.push(id))
        task.waitFor?.forEach(id => queue.push(id))
        // ... 其他关系
      }
    }
    depth++
  }

  return Array.from(related)
}
```

## 📚 与现有功能的关系

### 图谱 vs 标签浏览器

| 特性 | 图谱 | 标签浏览器 |
|------|------|-----------|
| 用途 | 可视化任务关系 | 按标签过滤任务 |
| 展示 | 图形化网络 | 树形列表 |
| 交互 | 拖拽、缩放、点击 | 展开、折叠、点击 |
| 适用 | 理解任务关联 | 快速查找任务 |

### 互补使用

**推荐方式**：
- **标签浏览器**：日常快速过滤和查找
- **关系图谱**：理解复杂项目的任务关系

**示例**：
```
1. 使用标签浏览器找到 #work/project-a 的所有任务
2. 打开某个任务详情
3. 点击"查看关系图谱"了解它与其他任务的关系
```
