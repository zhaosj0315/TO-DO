## 📖 功能概述

演示模式（Tutorial Mode）是一个交互式新手引导系统，通过高亮提示 + 遮罩层 + 步骤引导的方式，帮助新用户快速了解应用的核心功能和使用方法。

**核心特性**:
- ✅ 新用户首次登录自动触发
- ✅ 手动点击 💡 按钮启动
- ✅ 用户级别的完成状态记录
- ✅ 跳过后不再自动触发
- ✅ **21 步完整教程**（覆盖 90% 功能）
- ✅ **结构化描述**（操作步骤 + 使用场景 + 小提示）
- ✅ **进度条显示**（实时显示完成百分比）
- ✅ **分类标签**（介绍/基础/进阶）

## 🎯 触发机制

### 1. **自动触发（新用户）**
- **时机**: 用户首次登录后 1 秒
- **条件**: 检查 `tutorial_{username}` 标记不存在
- **效果**: 自动弹出演示模式，引导用户了解功能

### 2. **手动触发（所有用户）**
- **位置**: 右上角 💡 按钮
- **效果**: 随时可以重新查看教程
- **适用**: 老用户想复习功能时使用

### 3. **完成状态记录**
```javascript
// 完成教程
tutorial_{username} = 'true'

// 跳过教程
tutorial_{username} = 'skipped'
```

## 💻 技术实现

### 自动触发逻辑
```javascript
// TodoView.vue - onMounted
onMounted(async () => {
  // ... 其他初始化代码
  
  // 检查是否需要自动启动演示模式（新用户首次登录）
  const { value: tutorialCompleted } = await Preferences.get({ 
    key: `tutorial_${userStore.currentUser}` 
  })
  
  if (!tutorialCompleted) {
    // 延迟1秒启动，让用户先看到主界面
    setTimeout(() => {
      showTutorial.value = true
    }, 1000)
  }
})
```

### 完成/跳过处理
```javascript
// 跳过教程
const handleTutorialSkip = async () => {
  showTutorial.value = false
  await Preferences.set({ 
    key: `tutorial_${userStore.currentUser}`, 
    value: 'skipped' 
  })
}

// 完成教程
const handleTutorialFinish = async () => {
  showTutorial.value = false
  await Preferences.set({ 
    key: `tutorial_${userStore.currentUser}`, 
    value: 'true' 
  })
  showNotification('🎉 恭喜完成新手教程！', 'success')
}
```

### 组件结构
```
TutorialMode.vue
├── 遮罩层 (.tutorial-mask)
├── 高亮区域 (.tutorial-highlight)
└── 提示卡片 (.tutorial-card)
    ├── 头部 (步骤进度 + 关闭按钮)
    ├── 内容 (图标 + 标题 + 描述)
    ├── 操作按钮 (上一步/下一步/完成)
    └── 跳过按钮
```

### 核心逻辑
```javascript
// 步骤配置
steps = [
  {
    target: '.selector',  // 目标元素选择器
    icon: '💡',           // 图标
    title: '标题',        // 标题
    description: '描述',  // 描述
    position: 'bottom'    // 卡片位置
  }
]

// 位置计算
updatePositions() {
  1. 获取目标元素位置 (getBoundingClientRect)
  2. 计算高亮区域位置和大小
  3. 计算提示卡片位置（避免超出屏幕）
  4. 应用样式
}
```

## 🔧 使用方法

### 新用户体验流程
1. 用户注册新账号
2. 登录成功，进入主界面
3. 1秒后自动弹出演示模式
4. 用户可以选择：
   - **跟随教程**: 逐步了解所有功能
   - **跳过教程**: 直接开始使用（不再自动触发）

### 老用户重新查看
1. 点击右上角 💡 按钮
2. 重新进入演示模式
3. 随时可以跳过或完成

### 开发者集成
```vue
<!-- 1. 导入组件 -->
import TutorialMode from '../components/TutorialMode.vue'

<!-- 2. 添加到模板 -->
<TutorialMode
  :active="showTutorial"
  @close="showTutorial = false"
  @finish="handleTutorialFinish"
/>

<!-- 3. 添加触发按钮 -->
<button @click="startTutorial">💡</button>
```

## 🐛 已知问题

1. **目标元素不存在**: 如果用户没有任务，步骤5-7会找不到目标元素
   - **解决方案**: 添加默认位置，或跳过该步骤

2. **窗口大小变化**: 调整窗口大小时，高亮位置可能不准确
   - **解决方案**: 已添加 resize 监听器

3. **动态内容**: 如果目标元素是动态加载的，可能定位失败
   - **解决方案**: 使用 nextTick 等待 DOM 更新
