# 任务提醒功能实现说明

## 功能概述

为 TO-DO App 添加了完整的任务提醒功能，用户可以为任务设置自定义提醒时间和提醒方式。

## 新增功能

### 1. 提醒设置
- ✅ **启用/禁用提醒**：通过复选框控制
- ✅ **自定义提醒时间**：精确到分钟的日期时间选择器
- ✅ **三种提醒方式**：
  - 静音：只显示通知，无声音无震动
  - 标准：通知 + 声音 + 震动
  - 强提醒：持续通知（不可滑动关闭）+ 声音 + 震动

### 2. 数据结构扩展
任务对象新增字段：
```javascript
{
  enableReminder: Boolean,      // 是否启用提醒
  reminderTime: String,         // 提醒时间（ISO格式）
  reminderSound: String         // 提醒方式：'silent' | 'default' | 'alarm'
}
```

### 3. UI 改进
- 在任务创建表单中添加提醒设置区域
- 任务列表中显示 🔔 图标标识已设置提醒的任务
- 悬停显示具体提醒时间

### 4. 通知调度
- 创建任务时自动调度通知
- 完成任务时自动取消通知
- 删除任务时自动取消通知
- 提醒时间验证（不能设置过去的时间）

## 技术实现

### 核心函数

#### scheduleTaskReminder(task)
调度任务提醒通知，使用 Capacitor LocalNotifications API。

**特性**：
- 时间验证（过去时间提示）
- 根据提醒方式配置通知属性
- 强提醒模式使用 `ongoing: true` 和 `autoCancel: false`

#### cancelTaskReminder(taskId)
取消指定任务的提醒通知。

**调用时机**：
- 任务完成时
- 任务删除时

### 通知配置

```javascript
// 静音模式
{
  sound: null,
  vibrate: false
}

// 标准模式
{
  sound: 'default',
  vibrate: true
}

// 强提醒模式
{
  sound: 'default',
  vibrate: true,
  ongoing: true,        // 持续通知
  autoCancel: false     // 不可滑动关闭
}
```

## 使用说明

### 创建带提醒的任务

1. 点击"展开"按钮显示任务创建表单
2. 输入任务名称
3. 勾选"🔔 启用提醒"
4. 选择提醒时间（必须是未来时间）
5. 选择提醒方式（静音/标准/强提醒）
6. 点击 ✓ 提交

### 查看提醒状态

- 已设置提醒的任务会在标题右侧显示 🔔 图标
- 鼠标悬停在图标上可查看具体提醒时间

### 取消提醒

- 完成任务会自动取消提醒
- 删除任务会自动取消提醒

## 注意事项

1. **时间限制**：提醒时间必须是未来时间，否则不会调度通知
2. **权限要求**：需要用户授予通知权限（应用启动时自动请求）
3. **强提醒模式**：使用 `ongoing: true` 可能在某些设备上表现不同
4. **通知ID**：使用任务ID作为通知ID，确保一对一映射

## 兼容性

- ✅ Android 5.0+
- ✅ 使用 Capacitor LocalNotifications 8.x
- ✅ 兼容现有的逾期提醒和每日摘要功能
