## 问题排查

### 问题 1: 演示模式不自动触发
**可能原因**:
- 用户已经完成过教程
- 延迟时间未到（等待 1 秒）
- 数据读取失败

**排查方法**:
1. 检查本地存储中的 `tutorial_{username}` 值
2. 查看控制台是否有错误
3. 确认 onMounted 是否正常执行

### 问题 2: 高亮区域位置不准确
**可能原因**:
- 目标元素未渲染完成
- CSS 样式影响定位
- 窗口大小变化

**排查方法**:
1. 检查目标元素是否存在
2. 查看 getBoundingClientRect 返回值
3. 确认 updatePositions 是否正常调用

### 问题 3: 完成状态未保存
**可能原因**:
- Preferences API 调用失败
- 用户名获取错误
- 异步操作未完成

**排查方法**:
1. 查看控制台错误
2. 确认 userStore.currentUser 是否正确
3. 检查 Preferences.set 是否成功
