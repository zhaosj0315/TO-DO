# 🤖 AI问答功能 - 快速开始

## 功能位置

在应用右上角，演示模式按钮（💡）的**左侧**，有一个紫色渐变的🤖按钮。

```
[🤖 AI] [💡 演示] [⟳ 刷新] [🗑️ 回收站] [👤 头像]
```

## 快速测试

### 1. 启动应用
```bash
npm run dev
# 或使用测试脚本
./test-ai-chat.sh
```

### 2. 打开AI问答
- 登录应用
- 点击右上角 🤖 按钮
- 弹出AI问答对话框

### 3. 配置模型

#### 选项1: 盘古大模型（推荐）
```
模型: 盘古大模型
API地址: http://localhost:8000/api/chat
```

#### 选项2: OpenAI
```
模型: OpenAI
API Key: sk-your-api-key
```

#### 选项3: 本地模型
```
模型: 本地模型
地址: http://localhost:11434
```

### 4. 开始提问

#### 方式1: 点击快捷问题
- 今天完成了什么？
- 本周完成情况
- 高优先级待办
- 效率分析
- 阻碍统计
- 管理建议

#### 方式2: 输入自定义问题
```
今天我做了什么？
本周完成了哪些任务？
有哪些高优先级的待办任务？
我遇到了哪些阻碍？
给我一些任务管理建议
```

## 功能特点

### ✅ 完整数据访问
AI可以看到：
- 所有任务（标题、描述、状态、分类、优先级）
- 执行日志（开始、进展、阻碍、方案、里程碑、完成）
- 番茄钟历史
- 时间统计（今日/本周/本月）
- 回收站任务

### ✅ 智能分析
AI可以：
- 统计任务完成情况
- 分析执行效率
- 识别阻碍和解决方案
- 提供管理建议
- 回答各种维度的查询

### ✅ 多模型支持
- 盘古大模型（华为）
- OpenAI（gpt-3.5-turbo）
- 本地模型（Ollama/llama2）

## 技术说明

### 数据格式
AI接收的是结构化的Markdown格式数据，包含：
- 整体统计
- 时间维度统计（今日/本周/本月）
- 所有任务详情（包括日志和番茄钟）
- 回收站任务

### API调用
```javascript
// 盘古大模型
POST http://localhost:8000/api/chat
{
  "messages": [
    { "role": "system", "content": "完整的任务数据..." },
    { "role": "user", "content": "用户问题" }
  ],
  "stream": false
}
```

### 配置持久化
所有配置自动保存到浏览器localStorage：
- `ai_api_type`: 模型类型
- `ai_pangu_url`: 盘古API地址
- `ai_api_key`: OpenAI API Key
- `ai_local_url`: 本地模型地址

## 常见问题

### Q: 为什么没有响应？
A: 检查：
1. API地址是否正确
2. 网络是否通畅
3. API服务是否运行
4. 浏览器控制台是否有错误

### Q: 回答不准确？
A: 确认：
1. 任务数据是否正确
2. 模型是否正确选择
3. API配置是否正确

### Q: 如何切换模型？
A: 在对话窗口顶部的下拉菜单中选择不同模型。

### Q: 数据会被保存吗？
A: 不会。数据仅在对话时临时发送给AI，不会永久存储。
