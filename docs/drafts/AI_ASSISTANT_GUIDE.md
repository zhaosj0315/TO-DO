# AI 智能助手使用指南

## 🎯 一个对话，两种能力

打开 🤖 AI 任务助手，直接对话即可，AI 会自动识别你的意图。

---

## 💬 使用示例

### 1️⃣ 查询任务
```
你：今天完成了什么？
AI：今日完成了 5 个任务...

你：本周效率如何？
AI：本周完成率 80%...

你：有哪些逾期任务？
AI：当前有 2 个逾期任务...
```

### 2️⃣ 创建任务
```
你：明天上午开会，下午写报告，晚上复习英语

AI：✨ 已提取 3 个任务：
    1. 开会（工作/高）
    2. 写报告（工作/中）
    3. 复习英语（学习/中）
    💡 回复 "确认创建" 来添加

你：创建

AI：✅ 已成功创建 3 个任务！
```

### 3️⃣ 混合使用
```
你：今天完成了什么？
AI：[显示完成任务]

你：帮我创建明天的任务：开会、写文档
AI：[提取任务]

你：确认
AI：✅ 创建成功！

你：本周还有哪些待办？
AI：[显示待办任务]
```

---

## 🔍 创建任务触发词

AI 会自动识别以下情况为"创建任务"：

### 关键词
- `创建任务`、`添加任务`、`新建任务`
- `帮我创建`、`帮我添加`

### 时间词
- `明天要`、`今天要`、`本周要`
- `需要完成`、`要做`

### 动作词
- `开会`、`写报告`、`复习`
- `学习`、`工作`

### 多任务描述
- 包含逗号、顿号、分号
- 长度超过 20 字

---

## ✅ 确认创建

提取任务后，回复以下任一词语即可创建：
- `确认`
- `创建`
- `好的`
- `是的`
- `yes`
- `ok`

---

## 💡 使用技巧

1. **自然表达**：像和人聊天一样自然
2. **明确描述**：任务描述越清晰，AI 识别越准确
3. **分隔符**：多个任务用逗号、顿号分隔
4. **时间信息**：包含时间词（明天、今天）更容易识别
