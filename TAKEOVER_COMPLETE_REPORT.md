### 写入流程
```
用户操作 
  ↓
本地存储（Preferences + localStorage）← 主数据源（await）
  ↓
数据库同步（MySQL/SQLite）← 备份数据源（async，不阻塞）
```

```
新设备登录
  ↓

  ↓

  ↓
应用正常使用
```

## 🚀 使用指南

### 首次配置
1. 打开"数据库配置"
2. 选择MySQL，输入数据库信息
3. 测试连接 → 保存配置
4. 开启"接管模式"
5. 完成！所有数据自动同步

### 换设备恢复
1. 新设备安装应用
2. 登录账号
3. 打开"数据库配置"
4. 输入相同的数据库信息
5. 点击"从数据库恢复"
6. 所有数据自动恢复

### 多设备同步
1. 所有设备配置相同数据库
2. 所有设备开启"接管模式"
3. 任意设备操作，其他设备自动同步
4. 真正的云端体验
