# 甘特图功能设计方案

## 📊 功能定位
在标签浏览器和关系图谱之后，新增**甘特图视图**，用于可视化任务的时间规划和进度。

## 🎯 核心功能

### 1. 基础展示
- **时间轴**：横轴显示日期（今天、明天、本周、本月）
- **任务条**：每个任务显示为一个横条
  - 起始时间：任务创建时间或自定义开始时间
  - 结束时间：任务截止时间
  - 进度条：根据任务状态显示进度（待办 0%、进行中 50%、已完成 100%）
- **颜色编码**：
  - 🔴 高优先级：红色
  - 🟠 中优先级：橙色
  - 🔵 低优先级：蓝色
  - ✅ 已完成：绿色（半透明）

### 2. 交互功能
- **点击任务条**：打开任务详情
- **拖拽调整**：拖动任务条调整时间（可选）
- **缩放时间轴**：切换视图（日/周/月）
- **筛选**：按分类、优先级、笔记本筛选

### 3. 依赖关系显示
- **依赖箭头**：显示任务之间的依赖关系
- **关键路径**：高亮显示关键任务链

## 🛠️ 技术实现

### 1. 使用 ECharts
```javascript
import * as echarts from 'echarts'

// 甘特图配置
const option = {
  tooltip: {
    formatter: (params) => {
      const task = params.data
      return `
        <strong>${task.name}</strong><br/>
        开始：${task.startTime}<br/>
        结束：${task.endTime}<br/>
        进度：${task.progress}%
      `
    }
  },
  grid: {
    left: 150,
    right: 50,
    top: 50,
    bottom: 50
  },
  xAxis: {
    type: 'time',
    min: startDate,
    max: endDate
  },
  yAxis: {
    type: 'category',
    data: taskNames
  },
  series: [
    {
      type: 'custom',
      renderItem: renderGanttItem,
      data: ganttData
    }
  ]
}
```

### 2. 数据结构
```javascript
const ganttData = computed(() => {
  return taskStore.tasks
    .filter(t => t.customDate) // 只显示有截止时间的任务
    .map(task => ({
      name: task.text,
      value: [
        task.created_at,           // 开始时间
        task.customDate,           // 结束时间
        task.text                  // 任务名称
      ],
      itemStyle: {
        color: getPriorityColor(task.priority),
        opacity: task.status === 'completed' ? 0.5 : 1
      },
      progress: getTaskProgress(task),
      taskId: task.id
    }))
})
```

## 📊 数据要求

### 必需字段
- `created_at`：任务创建时间（开始时间）
- `customDate`：任务截止时间（结束时间）
- `text`：任务名称
- `priority`：优先级
- `status`：状态

### 可选字段
- `progress`：任务进度（0-100）
- `waitFor`：依赖的任务 ID
- `category`：分类
- `collectionId`：所属笔记本

## 🎯 使用场景

### 1. 项目管理

- 识别时间冲突
- 调整任务优先级

### 2. 时间规划

### 3. 团队协作
- 展示任务依赖关系
- 识别关键路径
- 优化资源分配

## 📝 注意事项

1. **性能优化**：
   - 只渲染可见时间范围内的任务
   - 虚拟滚动（任务数 > 100）
   - 防抖处理拖拽事件

2. **数据完整性**：
   - 没有截止时间的任务不显示
   - 提示用户设置截止时间

3. **用户体验**：
   - 加载动画
   - 错误提示
   - 操作引导
