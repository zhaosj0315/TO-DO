# TO-DO App v1.6.11 发布说明

**发布日期**: 2026-02-23  
**版本类型**: 功能增强版本

---

## 🎉 核心亮点

### 1. 多时间维度筛选功能
全新的时间筛选体验，让您从不同角度查看任务：
- **创建时间筛选**: 查看某个时间段创建的所有任务
- **截止时间筛选**: 查看某个时间段需要完成的任务
- **完成时间筛选**: 查看某个时间段实际完成的任务

### 2. 完成时间记录系统
准确记录每个任务的完成时刻：
- 点击复选框时自动记录完成时间
- 已完成任务按完成时间倒序排列
- 实际耗时计算更准确

### 3. 分页器紧凑化
更适合手机端的分页设计：
- 从4个箭头简化为2个
- 单击翻页，双击跳首/尾页
- 统一12px字体，圆形边框更清晰

### 4. 数据安全保障系统
多重提醒确保数据安全：
- 登录页醒目备份警告
- 首次登录欢迎弹窗
- 定期备份提醒（30天或任务>50且7天）
- Excel导出17个完整字段

---

## 📊 详细更新

### 新增功能

#### 多时间维度筛选
- 在高级筛选弹窗中新增3个时间维度按钮
- 激活状态显示紫色渐变背景
- Store层动态计算截止时间
- 支持与其他筛选条件组合使用

#### 完成时间记录
- `completed_at`字段准确记录完成时刻
- 取消完成时自动删除该字段
- 导入Excel时正确设置已完成任务的完成时间
- 只在有完成时间时显示实际耗时

#### 分页器优化
- 删除首页/末页按钮（‹‹ ››）
- 保留上一页/下一页按钮（‹ ›）
- 左箭头：单击上一页，双击跳首页
- 右箭头：单击下一页，双击跳末页
- 删除Go按钮，输入框回车或失焦自动跳转
- 统一字体为12px，箭头加粗显示
- 添加1.5px圆形边框，尺寸增至28px

#### 数据安全保障
- 登录页添加黄色备份警告框
- 首次登录显示欢迎弹窗（只显示一次）
- 数据管理区域强化警告提示
- 定期备份提醒：
  - 30天未备份时提醒
  - 任务>50且7天未备份时提醒
- 导出成功自动记录备份时间

### 功能优化

#### 已完成任务排序
- 按`completed_at`倒序排列
- 最近完成的任务显示在最前面
- 无`completed_at`时使用`created_at`作为备用

#### Excel导出完善
- 修复网页版导出路径问题
- 完善导出字段至17个：
  - 任务名称、描述、类型、分类、优先级、状态
  - 创建时间、截止时间、计划完成时间、实际完成时间
  - 实际耗时、番茄数、时长/规模
  - 指定日期、指定时间、重复周期、任务ID
- 增加导出数据容错处理
- 时长/规模字段增加详细说明

#### 番茄计算逻辑统一
- 全局统一番茄计算逻辑（22处修复）
- 个人主页番茄统计使用新规则
- 数据报告番茄统计使用新规则
- 所有统计传入完整任务对象

#### 页脚功能增强
- 添加使用指南弹窗
- 添加番茄规则弹窗
- 反馈弹窗更新（添加GitHub Issues链接）

## 🔧 技术改进

### 数据结构
新增任务字段：
```javascript
{
  completed_at: String,    // 实际完成时间（ISO格式）
  duration: String,        // 时长选择（快速/正常/较长）
  scale: String,           // 规模选择（小型/中型/大型）
}
```

### Store API
新增方法：
- `calculateDeadline(task)`: 计算任务截止时间
- 支持多时间维度的`getFilteredTasks()`

### 性能优化
- 优化大数据量下的筛选性能
- 优化分页器渲染性能
