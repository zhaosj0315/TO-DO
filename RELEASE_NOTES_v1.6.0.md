# TODO App v1.6.0 Release Notes

**发布日期**: 2026-02-21  
**版本**: v1.6.0

---

## 🎉 重大更新

### UI 视觉系统全面升级
- **统计区域主次分明**: 核心指标（全部、已逾期）宽度+20%，更加醒目
- **色彩语义化**: 已完成(淡绿)、待办(淡蓝)、已逾期(淡红)背景
- **微交互增强**: 悬停上浮2px，阴影加深
- **创建任务区凹陷感设计**: 输入框内阴影引导点击
- **右上角胶囊化封装**: iOS 风格半透明容器，毛玻璃效果
- **任务列表标签严格对齐**: 统一24px高度，垂直中心线对齐
- **高级筛选弹窗紧凑化**: 模块间距压缩44%，信息密度提升

### 交互优化
- **刷新按钮功能优化**: 点击刷新按钮重置所有筛选条件到初始状态
- **筛选和展开按钮优化**: 添加中文注释（"筛选"/"展开"/"收起"），降低学习成本

## 📦 下载安装

### Android 版本
**文件**: `TODO-App-1.6.0-Android.apk` (4.6 MB)

**安装步骤**:
1. 下载 APK 文件
2. 在 Android 设备上开启"允许安装未知来源应用"
3. 安装 APK 文件
4. 打开应用并注册新账号
5. 开始离线管理您的任务！

**系统要求**: Android 5.0+

---

### Mac 版本
**文件**: `TODO-App-1.6.0-mac.dmg` (123 MB)

**安装步骤**:
1. 下载 DMG 文件
2. 双击打开 DMG
3. 将 TODO App 拖入 Applications 文件夹
4. 从启动台或应用程序文件夹打开
5. 首次打开可能需要在"系统偏好设置 > 安全性与隐私"中允许运行

**系统要求**: macOS 10.13+  
**支持架构**: Intel (x64) + Apple Silicon (arm64)

---

### iOS 模拟器版本（开发者专用）
**文件**: `TODO-App-1.6.0-iOS-Simulator.zip` (1.8 MB)

**使用方法**:
1. 解压 ZIP 文件
2. 打开 Xcode 模拟器
3. 将 App.app 拖入模拟器窗口
4. 或使用命令: `xcrun simctl install booted App.app`

**注意**: 仅适用于 Mac 上的 iOS 模拟器，无法安装到真实 iPhone 设备

## ✨ 核心功能

### 用户系统
- 本地注册与登录
- 动态首字母渐变头像
- 手机号绑定与验证码登录
- 用户数据完全隔离

### 任务管理
- **7种任务类型**: 今天、明天、本周内、指定日期、每天重复、工作日重复、每周重复
- **3种分类**: 工作💼、学习📚、生活🏠
- **3种优先级**: 高、中、低
- **截止时间预警**: 颜色分级提醒（蓝色/橙色/红色）
- **番茄钟系统**: 根据优先级获得番茄奖励

### 智能筛选
- 按状态筛选（全部/已完成/待办/已逾期）
- 按分类筛选
- 按优先级筛选
- 按日期范围筛选
- 关键字搜索（模糊匹配）
- 所有筛选条件可组合使用

### 数据管理
- 导出任务到 Excel
- 从 Excel 批量导入任务
- 回收站功能（软删除）
- 完全离线运行，无需网络

## 🔒 隐私保护

- ✅ **完全离线**: 不联网，不收集任何用户信息
- ✅ **本地存储**: 所有数据存储在设备本地
- ✅ **数据隔离**: 多用户数据完全隔离
- ⚠️ **备份提示**: 卸载应用会丢失数据，请定期导出备份
