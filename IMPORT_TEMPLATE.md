# 任务导入模板说明

## 📋 导入格式

导入文件必须是 **Excel格式** (.xlsx 或 .xls)，包含以下列：

| 列名 | 是否必填 | 可选值 | 说明 |
|------|---------|--------|------|
| 任务名称 | ✅ 必填 | 任意文本 | 任务的标题 |
| 详细描述 | ❌ 可选 | 任意文本 | 任务的详细说明 |
| 分类 | ❌ 可选 | 工作 / 学习 / 生活 | 默认：工作 |
| 优先级 | ❌ 可选 | 高 / 中 / 低 | 默认：中 |
| 类型 | ❌ 可选 | 仅今天 / 每天 / 每周: 周一, 周二... | 默认：仅今天 |
| 状态 | ❌ 可选 | 待办 / 已完成 / 已逾期 | 默认：待办 |
| 创建时间 | ❌ 可选 | YYYY-MM-DD HH:mm:ss | 默认：当前时间 |

## 🎯 使用步骤

1. **导出现有任务**（推荐）
   - 打开APP → 个人主页 → 数据管理 → 导出任务
   - 这会生成一个标准格式的Excel文件作为模板

2. **编辑Excel文件**
   - 使用Excel、WPS或Google Sheets打开
   - 按照上述格式添加或修改任务
   - 确保列名完全一致（包括中文字符）

3. **导入任务**
   - 打开APP → 个人主页 → 数据管理 → 导入任务
   - 选择编辑好的Excel文件
   - 等待导入完成提示

## ⚠️ 注意事项

1. **必填字段**：只有"任务名称"是必填的，其他字段可以留空
2. **格式严格**：列名必须完全匹配（包括中文字符）
3. **数据验证**：
   - 分类只能是：工作、学习、生活
   - 优先级只能是：高、中、低
   - 类型只能是：仅今天、每天、每周格式
   - 状态只能是：待办、已完成、已逾期
4. **周期格式**：
   - 每周任务格式：`每周: 周一, 周三, 周五`
   - 周几之间用中文逗号分隔
5. **时间格式**：
   - 标准格式：`2026-02-18 10:30:00`
   - 也支持：`2026-02-18` 或 `2026/02/18`
6. **导入结果**：
   - 成功和失败的数量会在导入后显示
   - 失败的行通常是因为任务名称为空或格式错误

## 💡 最佳实践

1. **先导出再导入**：使用导出功能生成标准模板
2. **小批量测试**：先导入几条测试数据，确认格式正确
3. **备份数据**：导入前先导出现有任务作为备份
4. **检查格式**：确保Excel中没有合并单元格或特殊格式
5. **编码问题**：使用UTF-8编码保存，避免中文乱码
