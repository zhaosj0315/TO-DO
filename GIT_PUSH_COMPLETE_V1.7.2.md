## 🐛 Bug 修复

### 1. 演示模式第一步闪现问题
- **问题**: 第一步时卡片从右下角闪现到中央
- **原因**: `cardStyle` 初始值为空对象
- **修复**: 初始化为居中样式
  ```javascript
  const cardStyle = ref({
    top: '50%',
    left: '50%',
    transform: 'translate(-50%, -50%)'
  })
  ```

### 2. 日志时间显示格式
- **问题**: 本年内只显示月/日，跨天任务看不出完整日期
- **修复**: 始终显示完整 `年/月/日 时:分` 格式
- **示例**: `2026/2/25 10:31`

### 3. 日志排序顺序
- **问题**: 日志按时间倒序显示（从晚到早）
- **修复**: 改为正序显示（从早到晚）
- **影响**: 任务卡片和详情页的日志列表

### 4. 移动端边界检测
- **问题**: 演示模式卡片在手机上可能超出屏幕边界
- **修复**: 
  - 响应式宽度：`min(400px, calc(100vw - 40px))`
  - 动态计算实际宽度
  - 严格边界保护（上下左右各20px）

```css
  width: min(400px, calc(100vw - 40px));

```

### 动态边界检测
```javascript
const actualCardWidth = Math.min(maxCardWidth, window.innerWidth - 40)

// 严格边界保护
if (left < margin) left = margin
if (left + actualCardWidth > window.innerWidth - margin) {
  left = window.innerWidth - actualCardWidth - margin
}
```
