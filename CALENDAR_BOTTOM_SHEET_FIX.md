# 日历视图 Bottom Sheet 布局修复报告

**修复日期**: 2026-03-12  
**版本**: v0.9.1+

## 📋 修复内容

### 1. CalendarView.vue 布局重构

#### ✅ 修改前问题
- 使用居中弹窗布局（`align-items: center`）
- 宽度限制 90%，不符合 Bottom Sheet 规范
- 缺少顶部小横条和紫色渐变头部
- 动画效果为缩放（`scale`），不符合从底部滑出的设计

#### ✅ 修改后效果
- **Bottom Sheet 布局**: 从底部滑出，左右全屏
- **顶部小横条**: 40px 宽度，灰色圆角横条
- **紫色渐变头部**: 统一的紫色渐变标题栏（📅 日历视图）
- **滑出动画**: `translateY(100%)` → `translateY(0)`
- **圆角设计**: 顶部圆角 20px，底部直角
- **向上阴影**: `box-shadow: 0 -4px 20px rgba(0, 0, 0, 0.15)`

### 2. Android 返回手势支持

#### ✅ TodoView.vue 返回逻辑
在 `backButtonHandler` 中添加日历视图的返回处理：

```javascript
} else if (showCalendar.value) {
  console.log('✅ 关闭日历视图')
  showCalendar.value = false
  return
}
```

**插入位置**: 在 `showAddLogModal` 之后，`showTaskPreview` 之前

**优先级**: 第三层弹窗（最上层）
