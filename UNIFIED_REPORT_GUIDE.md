## 🎯 各章节数据来源

### 1. 智能总结
```javascript
`本期共完成 ${completedTasks} 个任务，完成率 ${completionRate}%，累计获得 ${totalPomodoros} 个番茄钟。`
```

### 2. 数据概览
- **完成任务**：时间范围内已完成的任务数
- **高优先级**：已完成的高优先级任务数
- **番茄钟**：已完成任务的番茄钟总和
- **分类统计**：按工作/学习/生活分类统计

### 3. 重点任务 (Top 10)
- **排序规则**：按番茄钟数降序
- **数据来源**：时间范围内已完成的任务
- **显示内容**：任务名称、分类、优先级、番茄钟数、完成时间、描述

### 4. 本期目标
- **数据来源**：时间范围内已完成的高优先级任务
- **显示数量**：Top 5
- **显示内容**：任务名称、分类、番茄钟数、描述

### 5. 风险与问题
- **数据来源**：当前所有逾期任务（不限时间范围）
- **显示数量**：Top 10
- **显示内容**：任务名称、分类、优先级、描述
- **改进建议**：
  - 建议优先处理逾期任务，避免影响整体进度
  - 可以考虑将大任务拆分为小任务，提高完成率
  - 合理安排时间，避免任务堆积

### 6. 下期计划
- **数据来源**：当前所有待办任务（不限时间范围）
- **排序规则**：按优先级排序（高 > 中 > 低）
- **显示数量**：Top 10
- **显示内容**：任务名称、分类、优先级、番茄钟数、描述

## 📱 操作流程

### 1. 生成报告
```
1. 打开个人中心
2. 点击"📊 智能报告中心"
3. 选择报告类型（自动生成）
   - 或选择"自定义"手动选择日期
4. 等待生成（显示加载动画）
5. 查看报告
```

### 2. 切换视图
```
1. 点击"📈 可视化"查看图表
2. 点击"📝 文本"查看结构化报告
3. 自由切换，数据实时同步
```

### 3. 保存报告
```
1. 点击"💾 保存"按钮
2. 报告保存到 localStorage
3. 显示成功提示
4. 可在"📚 报告历史"中查看
```

### 4. 导出报告
```
1. 点击"📄 导出"导出HTML（开发中）
2. 点击"📋 复制"复制纯文本
3. 粘贴到其他应用使用
```

### 5. 查看历史
```
1. 点击"📚 历史"按钮
2. 查看所有保存的报告
3. 点击历史报告重新打开
```

## 🔧 技术实现

### 组件结构
```
UnifiedReportModal.vue (主容器)
├── 报告类型选择器
├── 自定义日期选择器
├── 视图切换器
├── LoadingSpinner (加载动画)
├── VisualReportView (可视化视图)
│   ├── Hero Section
│   ├── AI摘要
│   ├── 核心数字
│   ├── 分类统计
│   └── 优先级分布
└── TextReportView (文本视图)
    ├── 报告周期
    ├── 智能总结
    ├── 数据概览
    ├── 重点任务
    ├── 本期目标
    ├── 风险与问题
    └── 下期计划
```

### 数据生成流程
```javascript
1. calculateDateRange()
   └── 根据报告类型计算起止日期

2. generateUnifiedReport(startDate, endDate)
   ├── 筛选时间范围内的任务
   ├── calculateBaseData() - 计算基础数据
   ├── generateVisualData() - 生成可视化数据
   └── generateTextData() - 生成文本数据

3. 展示报告
   ├── visualData → VisualReportView
   └── textData → TextReportView
```

### 关键函数

#### calculateDateRange()
- 功能：根据报告类型计算日期范围
- 输入：selectedType (报告类型)
- 输出：{ startDate, endDate }

#### generateUnifiedReport()
- 功能：生成统一报告数据
- 输入：startDate, endDate
- 输出：{ visualData, textData }

#### calculateBaseData()
- 功能：计算基础统计数据
- 输入：tasks, startDate, endDate
- 输出：基础数据对象

#### generateVisualData()
- 功能：生成可视化数据
- 输入：baseData, tasks
- 输出：可视化数据对象

#### generateTextData()
- 功能：生成文本报告数据
- 输入：baseData, tasks, startDate, endDate
- 输出：文本数据对象（包含所有章节）
