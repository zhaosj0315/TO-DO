## 🎉 版本亮点

### AI 对话历史记录系统
全新的多对话管理功能，让 AI 助手更像 ChatGPT！

**核心特性**:
- 📚 **多对话管理**: 支持创建、切换、删除多个独立对话
- 📂 **智能分组**: 自动按时间分组（今天/昨天/最近7天/更早）
- 💾 **数据持久化**: 所有对话历史自动保存到本地
- 🎨 **精美界面**: 左侧侧边栏 + 渐变激活效果

## ✨ 新增功能

### 1. 对话历史侧边栏
- **位置**: AI 问答界面左侧
- **宽度**: 260px，可折叠
- **功能**: 
  - 显示所有历史对话
  - 点击切换对话
  - 悬停显示删除按钮

### 2. 时间智能分组
对话自动按时间分组显示：
- **今天**: 显示具体时间（如 14:30）
- **昨天**: 显示"昨天"
- **最近7天**: 显示"X天前"
- **更早**: 显示日期（如 2/20）

### 3. 新建对话
- **按钮**: ➕ 新对话（紫色渐变）
- **位置**: 侧边栏顶部
- **功能**: 一键创建新对话，自动切换

### 4. 对话标题自动生成
- **规则**: 取首条用户消息作为标题
- **长度**: 自动截断过长标题
- **显示**: 支持省略号显示

### 5. 激活状态视觉增强
当前对话高亮显示：
- 紫色边框
- 渐变背景（#f0f4ff → #faf5ff）
- 阴影效果
- 标题文字变紫色加粗

## 💾 数据存储

### localStorage 结构
```javascript
// 对话列表
ai_chat_list: [
  {
    id: "1709012345678",
    title: "今天完成了什么？",
    messages: [...],
    createdAt: "2026-02-26T14:30:00.000Z",
    updatedAt: "2026-02-26T15:30:00.000Z"
  },
  ...
]

// 当前对话 ID
ai_current_chat_id: "1709012345678"
```

### 数据持久化机制
- **自动保存**: 每次发送消息后自动保存
- **自动加载**: 打开 AI 问答时自动加载最近对话
- **数据隔离**: 每个对话独立存储，互不影响

## 📱 使用指南

### 创建新对话
1. 点击 AI 问答按钮（🤖）
2. 点击左上角"➕ 新对话"按钮
3. 开始提问

### 切换对话
1. 在左侧历史记录中找到目标对话
2. 点击对话项
3. 右侧自动加载对话内容

### 删除对话
1. 悬停在目标对话上
2. 点击右侧出现的🗑️按钮
3. 确认删除

### 折叠侧边栏
1. 点击右上角的◀按钮
2. 侧边栏折叠，对话区域扩大
3. 再次点击▶按钮展开

## 📊 性能优化

### 数据加载
- **懒加载**: 只加载当前对话的消息
- **缓存机制**: 已加载的对话缓存在内存中
- **增量更新**: 只更新变化的对话

### 渲染优化
- **虚拟滚动**: 历史记录列表支持大量对话
- **条件渲染**: 折叠时不渲染侧边栏内容
- **防抖处理**: 搜索和筛选操作防抖
