## 🎯 核心特性

### 智能特性
1. **自动进度计算**：子任务完成时自动更新父任务进度
2. **依赖状态同步**：前置任务完成时自动解锁后置任务
3. **循环依赖保护**：添加依赖前自动检测循环
4. **批量操作**：完成/删除父任务时自动处理所有子任务
5. **关键路径分析**：自动计算项目关键路径

### 灵活性
1. **3 层嵌套**：支持任务 → 子任务 → 孙任务
2. **继承规则**：子任务可继承父任务的截止时间/分类/标签
3. **独立优先级**：子任务可设置独立的优先级
4. **多对多依赖**：一个任务可以依赖多个前置任务

## 🚀 如何使用

### 立即体验（3 步）
```bash
# 1. 刷新页面（自动迁移数据）
# 2. 打开浏览器控制台（F12）
# 3. 执行测试代码（见 QUICK_START.md）
```

### 示例代码
```javascript
// 创建父任务
const parent = await taskStore.addTask({
  text: '开发新功能',
  type: 'custom_date',
  category: 'work',
  priority: 'high',
  customDate: '2026-03-01'
})

// 添加子任务
await taskStore.addSubtask(parent.id, {
  text: '需求分析',
  type: 'today',
  category: 'work',
  priority: 'medium'
})

// 设置依赖
await taskStore.addDependency(task2.id, task1.id)
```
