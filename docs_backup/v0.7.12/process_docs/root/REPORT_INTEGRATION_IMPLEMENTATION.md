## 🚀 使用流程

### 用户操作步骤

1. **打开报告中心**
   - 个人中心 → 点击"📊 智能报告中心"

2. **选择报告类型**
   - 点击任意类型按钮（日报/周报/月报等）
   - 自动生成报告

3. **切换视图**
   - 点击"📈 可视化"查看图表
   - 点击"📝 文本"查看结构化报告

4. **保存或导出**
   - 点击"💾 保存"保存到历史
   - 点击"📄 导出"导出HTML
   - 点击"📋 复制"复制文本

5. **查看历史**
   - 点击"📚 历史"查看所有保存的报告

## 🔄 数据流程

```
用户选择报告类型
    ↓
calculateDateRange() - 计算日期范围
    ↓
generateUnifiedReport() - 生成统一报告
    ↓
├── calculateBaseData() - 计算基础数据
├── generateVisualData() - 生成可视化数据
└── generateTextData() - 生成文本数据
    ↓
展示报告（双视图）
    ↓
用户操作
├── 保存 → handleReportSaved() → localStorage
├── 导出 → exportHTML()
├── 复制 → copyText() → clipboard
└── 历史 → showReportHistory()
```
