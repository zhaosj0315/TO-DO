## 🚀 快速开始

### 安装应用

#### Android 设备
1. **下载安装包**: 获取 `TODO-App.apk` 文件
2. **允许安装**: 设置 → 安全 → 允许安装未知来源应用
3. **安装应用**: 点击 APK 文件完成安装
4. **首次启动**: 打开应用，进入注册界面

#### Windows 桌面端
1. **下载**: 获取 `TODO App Setup 0.8.9.exe` 安装程序
2. **安装**: 双击运行，选择安装目录
3. **启动**: 从桌面快捷方式或开始菜单启动

#### Mac 桌面端
1. **下载**: 获取 `TODO-App-0.8.9-mac.zip` 压缩包
2. **解压**: 双击解压到应用程序文件夹
3. **启动**: 从启动台或应用程序文件夹打开

### 创建账户

1. **注册新账户**:
   - 输入用户名（3-20个字符）
   - 设置密码（6位以上）
   - 可选：设置密保问题
   - 可选：绑定手机号

2. **登录方式**:
   - 用户名 + 密码
   - 手机号 + 验证码（如已绑定）

3. **个性化头像**:
   - 系统自动生成基于用户名首字母的渐变头像
   - 每个用户都有独特的颜色组合

### 创建第一个任务

1. **点击输入框**: 主界面顶部的任务输入框
2. **输入任务标题**: 例如"完成项目报告"
3. **选择任务属性**:
   - 📅 **时间**: 今天/明天/本周/指定日期/重复任务
   - 🏷️ **分类**: 工作💼/学习📚/生活🏠
   - ⚡ **优先级**: 高🔴/中🟡/低🔵
4. **点击提交**: 绿色 ✓ 按钮完成创建

## 📝 基础功能

### 任务类型详解

#### 1. 时间类型
- **今天**: 当天需要完成的任务
- **明天**: 明天的计划任务
- **本周内**: 本周任意时间完成
- **指定日期**: 选择具体的日期和时间
- **每天重复**: 每天都会出现的任务（如晨练）
- **工作日重复**: 周一到周五的重复任务（如上班）
- **每周重复**: 每周特定日子的任务（如周会）

#### 2. 分类系统
- **工作** 💼: 职业相关任务
- **学习** 📚: 教育培训任务
- **生活** 🏠: 日常生活任务

#### 3. 优先级系统
- **高优先级** 🔴: 紧急重要任务，显示4个🍅
- **中优先级** 🟡: 一般重要任务，显示2个🍅
- **低优先级** 🔵: 可延后任务，显示1个🍅

### 任务操作

#### 基本操作
- **完成任务**: 点击任务左侧的 ⭕ 圆圈
- **编辑任务**: 点击任务标题进入编辑模式
- **删除任务**: 长按任务 → 选择删除 → 移入回收站
- **置顶任务**: 点击 📌 按钮将重要任务置顶

#### 任务状态
- **待办** ⏳: 尚未完成的任务
- **已完成** ✅: 已经完成的任务
- **已逾期** ⚠️: 超过截止时间的任务

#### 截止时间显示
- **蓝色**: 时间充足（>3小时）
- **橙色**: 时间紧张（1-3小时）
- **红色**: 非常紧急（<1小时）
- **灰色**: 已逾期

### 智能排序系统

任务列表采用智能权重排序：
1. **置顶任务**: 永远在最前面
2. **逾期任务**: 按逾期时间排序
3. **紧急任务**: 2小时内到期自动提升
4. **普通任务**: 按优先级和创建时间排序
5. **已完成任务**: 显示在最后

## 📎 富媒体附件系统

### 支持的文件类型

#### 图片类型 📸
- **格式**: JPG, PNG, GIF, BMP, WebP
- **上传方式**: 🖼️ 图片按钮
- **预览**: 全屏居中显示，黑色背景
- **特性**: 默认300px宽度，点击可全屏预览

#### 文档类型 📄
- **PDF** 📕: 浏览器原生预览，支持缩放翻页
- **Word** 📄: DOC/DOCX格式，显示文件卡片
- **文本** 📄: TXT/RTF格式，等宽字体显示

#### 表格类型 📊
- **Excel**: XLS/XLSX/CSV格式
- **显示**: 文件卡片，点击下载

#### 演示类型 📽️
- **PowerPoint**: PPT/PPTX格式
- **显示**: 文件卡片，点击下载

#### 视频类型 🎬
- **格式**: MP4, MOV, AVI, MKV, FLV, WMV
- **预览**: HTML5播放器，支持播放控制

#### 压缩包类型 📦
- **格式**: ZIP, RAR, 7Z, TAR, GZ
- **显示**: 文件卡片，点击下载

### 上传文件操作

#### 上传图片
1. **进入编辑模式**: 点击任务输入框或编辑现有任务
2. **点击图片按钮**: 工具栏中的 🖼️ 图片按钮
3. **选择图片**: 从相册中选择图片
4. **自动插入**: 系统自动插入 `![图片](local://img_xxx)` 语法
5. **预览效果**: 切换到预览模式查看真实图片

#### 上传文件
1. **进入编辑模式**: 点击任务输入框或编辑现有任务
2. **点击文件按钮**: 工具栏中的 📎 文件按钮
3. **选择文件**: 从设备中选择文档、视频等文件
4. **大小检查**: 系统自动检查文件大小（限制10MB）
5. **自动插入**: 系统自动插入 `[📎 文件名](local://file_xxx)` 语法
6. **预览效果**: 切换到预览模式查看文件卡片

### 文件预览操作

#### 预览图片
1. **点击图片**: 在任务中点击任何图片
2. **全屏显示**: 自动打开全屏预览弹窗
3. **查看详情**: 显示文件名和大小信息
4. **下载图片**: 点击下载按钮保存到设备

#### 预览文档
1. **点击文件卡片**: 点击任务中的文件卡片
2. **预览内容**:
   - **PDF**: 在弹窗中直接预览，支持缩放和翻页
   - **视频**: HTML5播放器，支持播放、暂停、音量控制
   - **文本**: 等宽字体显示，保留原始格式
3. **下载文件**: 所有文件都支持一键下载

#### 不支持预览的文件
- **Word/Excel/PPT**: 显示友好提示和下载按钮
- **压缩包**: 显示文件信息和下载选项
- **其他格式**: 提供下载功能

### 文件管理

#### 存储位置
- **本地存储**: 所有文件保存在设备本地
- **存储路径**: `应用数据/media/{任务ID}/文件名`
- **离线访问**: 无需网络即可查看所有文件

#### 文件信息
- **文件名**: 显示原始文件名
- **文件大小**: 自动格式化显示（KB/MB）
- **上传时间**: 记录文件上传时间
- **文件类型**: 自动识别并显示对应图标

#### 注意事项
- **文件大小限制**: 单个文件不超过10MB
- **存储空间**: 依赖设备可用存储空间
- **文件安全**: 所有文件仅存储在本地，不会上传到云端

### AI任务拆分

#### 功能介绍
- **智能拆解**: 将复杂任务自动拆分为可执行的子任务
- **时间估算**: AI为每个子任务估算所需时间
- **依赖关系**: 自动设置子任务间的依赖关系

#### 使用步骤
1. **创建任务**: 输入任务标题和详细描述
2. **点击AI拆分**: 在任务详情页点击🤖AI拆分
3. **选择模板**: 
   - 快速拆分：3-5个主要步骤

   - 时间优先：按时间顺序拆分

4. **预览结果**: 查看AI生成的子任务列表
5. **编辑调整**: 可以修改、删除或添加子任务
6. **确认创建**: 一次性创建所有子任务

#### 智能提醒

- **智能提醒**: 每日9:00推送任务统计和激励语
- **完成建议**: 基于历史数据提供完成建议

## 📊 数据管理

### 数据导出

#### Excel导出

2. **点击导出**: 选择"📤 导出Excel"
3. **选择位置**: 系统自动选择最佳保存位置
4. **导出内容**: 包含所有任务、日志、番茄钟记录

#### 完整备份
1. **JSON格式**: 导出完整的JSON数据文件
2. **包含内容**: 
   - 所有任务数据
   - 执行日志记录
   - 番茄钟历史
   - 用户配置信息
   - AI对话历史

### 数据导入

#### Excel导入
1. **下载模板**: 点击"📥 下载模板"获取标准格式
2. **填写数据**: 按模板格式填写任务信息
3. **导入文件**: 选择"📥 导入Excel"上传文件
4. **数据验证**: 系统自动验证数据格式

- **使用模板**: 严格按照官方模板格式

#### 数据恢复

## 🔧 高级功能

### 任务依赖关系

#### 设置依赖
1. **编辑任务**: 进入任务详情页
2. **设置等待**: 点击"等待任务"选项
3. **选择依赖**: 从任务列表中选择要等待的任务
4. **保存设置**: 确认依赖关系

#### 依赖状态
- **等待中** 🔒: 显示黄色徽章，等待其他任务完成
- **被依赖** 🔓: 有其他任务在等待这个任务
- **无依赖** ✅: 可以立即开始的任务

#### 智能通知
- **自动解锁**: 依赖任务完成后自动通知等待的任务
- **任务跳转**: 点击等待任务可直接跳转到详情页

### 番茄钟系统

#### 专注计时
1. **启动番茄钟**: 点击任务卡片上的🍅按钮
2. **专注25分钟**: 全屏紫色界面，圆形进度条
3. **休息提醒**: 自动切换到5分钟短休息
4. **长休息**: 每4个番茄钟后15分钟长休息

#### 番茄钟控制
- **暂停/继续**: 专注过程中可以暂停
- **放弃**: 可以提前结束当前番茄钟
- **跳过休息**: 休息时可以跳过直接开始下一个
- **继续下一个**: 休息结束后继续下一个番茄钟

#### 统计功能
- **今日统计**: 查看今天的专注时长和完成数
- **本周统计**: 追踪本周的番茄钟完成情况
- **历史记录**: 查看所有番茄钟的详细记录
- **任务关联**: 每个番茄钟都关联到具体任务

### 任务执行日志

#### 日志类型
- **开始** 🚀: 记录任务开始时间和计划
- **进展** 📈: 记录任务进度和当前状态
- **阻碍** 🚫: 记录遇到的问题和困难
- **方案** 💡: 记录解决问题的方法
- **里程碑** 🎯: 记录重要的进展节点
- **完成** ✅: 记录任务完成和总结

#### 添加日志
1. **进入任务详情**: 点击任务标题
2. **点击添加日志**: 选择"➕ 添加日志"
3. **选择类型**: 从6种日志类型中选择
4. **填写内容**: 输入详细的日志内容
5. **设置属性**: 
   - 耗时：记录本次推进的时间
   - 进度：设置当前完成百分比
   - 心情：记录执行过程中的感受
   - 标签：添加自定义标签分类

#### 执行统计
- **推进次数**: 统计总共推进了多少次
- **总耗时**: 累计所有推进的时间
- **平均时长**: 计算每次推进的平均时间
- **阻碍统计**: 统计遇到和解决的问题数量

### 智能排序与筛选

#### 排序规则
1. **置顶任务**: 📌标记的任务永远在最前面
2. **逾期任务**: 按逾期时间长短排序
3. **紧急任务**: 2小时内到期自动提升优先级
4. **普通任务**: 按优先级和创建时间排序
5. **已完成任务**: 显示在列表最后

#### 高级筛选
1. **打开筛选**: 点击🔍筛选按钮
2. **设置条件**:
   - **状态**: 全部/待办/已完成/已逾期
   - **分类**: 工作/学习/生活
   - **优先级**: 高/中/低
   - **日期范围**: 自定义时间段
   - **关键词**: 模糊搜索任务标题和描述
3. **应用筛选**: 点击确定查看筛选结果
4. **重置筛选**: 点击重置按钮清除所有条件

## ❓ 常见问题

### 安装和启动

**Q: Android安装时提示"未知来源"怎么办？**
A: 进入设置 → 安全 → 允许安装未知来源应用，然后重新安装。

**Q: Windows版本无法启动怎么办？**
A: 确保系统已安装最新的Visual C++运行库，或以管理员身份运行。

**Q: Mac版本提示"无法验证开发者"怎么办？**
A: 右键点击应用 → 打开，或在系统偏好设置 → 安全性与隐私中允许。

### 数据和同步

**Q: 数据存储在哪里？**
A: 所有数据存储在设备本地，使用Capacitor Preferences API，不会上传到云端。

**Q: 换手机后数据会丢失吗？**
A: 是的，数据不会自动同步。建议定期导出备份，换设备后导入恢复。

**Q: 多个设备可以同步数据吗？**
A: 目前不支持自动同步，但可以通过导出/导入功能手动同步数据。

**Q: 卸载应用后数据还在吗？**
A: 卸载应用会清除所有本地数据，请务必在卸载前导出备份。

### 功能使用

**Q: 富媒体文件有大小限制吗？**
A: 单个文件限制10MB，主要考虑应用性能和存储空间。

**Q: 支持哪些图片格式？**
A: 支持JPG、PNG、GIF、BMP、WebP等常见图片格式。

**Q: AI功能需要网络吗？**
A: AI功能需要网络连接，但可以配置本地Ollama模型实现完全离线。

**Q: 重复任务如何工作？**
A: 重复任务完成后会在指定时间自动重置为待办状态，无需手动创建。

### 性能和兼容性

**Q: 应用运行缓慢怎么办？**
A: 建议定期清理已完成任务，或导出数据后重新开始。

**Q: 支持哪些Android版本？**
A: 支持Android 7.0（API 24）及以上版本。

**Q: 支持平板电脑吗？**
A: 支持，界面会自动适配不同屏幕尺寸。
