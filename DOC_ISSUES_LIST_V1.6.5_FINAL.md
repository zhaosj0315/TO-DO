### ⚠️ 问题4: USER_MANUAL.md 操作指南缺失
**状态**: 待修复  
**位置**: `/USER_MANUAL.md`  
**问题描述**: 缺少数据报告功能的详细操作指南  
**缺失内容**:
- 如何生成报告
- 如何选择报告类型
- 如何自定义日期范围
- 如何导出报告

**建议修复**:
```markdown
## 📊 数据报告功能

### 生成报告
1. 点击右上角"📊"图标
2. 选择报告类型（日报/周报/月报/季报/半年报/年报/自定义）
3. 如果选择"自定义"，选择开始和结束日期
4. 点击"生成报告"按钮

### 导出报告
1. 生成报告后，点击底部导出按钮
2. 选择导出格式：
   - **📄 导出HTML**: 完整样式 + 图表，可打印为 PDF
   - **📋 复制文本**: 复制纯文本到剪贴板
   - **📝 导出Markdown**: 下载 .md 文件

### 打印为 PDF
1. 导出 HTML 文件
2. 用浏览器打开 HTML 文件
3. 按 Cmd+P (Mac) 或 Ctrl+P (Windows)
4. 选择"另存为 PDF"
```
