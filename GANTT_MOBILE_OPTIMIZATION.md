# 甘特图移动端优化方案

**优化时间**: 2026-03-12 15:59  
**优化目标**: 提升甘特图在小屏设备上的显示效果  
**优化原则**: 响应式设计 + 横向滚动

---

## 🎯 问题分析

### 当前问题
1. ❌ 任务名称过长，Y轴占用空间太大
2. ❌ 图表区域太窄，时间轴显示不全
3. ❌ 控制按钮在小屏上拥挤
4. ❌ 统计信息排列不合理
5. ❌ 字体大小未适配移动端

### 影响
- 用户体验差
- 信息密度过高
- 操作不便

---

## ✅ 优化方案

### 1. 响应式布局

#### 任务名称截断
```javascript
// 桌面端：15字符
// 移动端：8字符
const isMobile = window.innerWidth < 768
const maxNameLength = isMobile ? 8 : 15
const taskNames = ganttData.value.map(t => 
  t.name.substring(0, maxNameLength) + (t.name.length > maxNameLength ? '...' : '')
)
```

#### Grid 边距调整
```javascript
grid: {
  left: isMobile ? 80 : 200,   // 移动端减少60%
  right: isMobile ? 20 : 60,   // 移动端减少67%
  top: 80,
  bottom: 50,
  containLabel: true
}
```

#### 字体大小适配
```javascript
xAxis: {
  axisLabel: {
    fontSize: isMobile ? 10 : 12  // 移动端缩小
  }
},
yAxis: {
  axisLabel: {
    fontSize: isMobile ? 11 : 13  // 移动端缩小
  }
}
```

### 2. 横向滚动支持

#### HTML 结构
```vue
<!-- 添加包裹层 -->
<div class="gantt-wrapper">
  <div ref="chartRef" class="gantt-container"></div>
</div>
```

#### CSS 样式
```css
.gantt-wrapper {
  flex: 1;
  overflow-x: auto;
  overflow-y: auto;
  -webkit-overflow-scrolling: touch; /* iOS 平滑滚动 */
}

.gantt-container {
  min-width: 600px; /* 移动端最小宽度 */
}
```

### 3. 头部优化

#### 统计信息纵向排列
```css
@media (max-width: 768px) {
  .gantt-stats {
    flex-direction: column;
    gap: 4px;
    font-size: 0.75rem;
  }
  
  .gantt-header h2 {
    width: 100%;
    order: -1; /* 标题移到第一行 */
  }
}
```

### 4. 控制按钮优化

#### 等宽平分
```css
@media (max-width: 768px) {
  .control-btn {
    flex: 1;           /* 等宽平分 */
    padding: 6px 10px; /* 减少内边距 */
    font-size: 0.8rem; /* 缩小字体 */
  }
}
```

### 5. 用户提示

#### 滑动提示
```vue
<div class="mobile-hint">💡 左右滑动查看完整时间轴</div>
```

```css
.mobile-hint {
  display: none; /* 桌面端隐藏 */
}

@media (max-width: 768px) {
  .mobile-hint {
    display: block; /* 移动端显示 */
  }
}
```
