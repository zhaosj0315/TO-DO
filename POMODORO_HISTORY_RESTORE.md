# 番茄钟历史记录功能恢复说明

## ✅ 已完成的修改

### 1. 在 EnhancedPomodoroStats.vue 中添加了番茄钟历史记录部分

**位置**：番茄钟标签页，分类占比之后

**功能**：
- 📜 按日期分组显示所有番茄钟记录
- 🔽 支持展开/折叠每天的记录（默认展开最近3天）
- ✅ 显示完成状态（完成/放弃）
- ⏱️ 显示开始时间、结束时间、时长
- 🔗 点击记录可跳转到对应任务详情

### 2. 新增的 computed 函数

```javascript
pomodoroHistoryByDate  // 按日期分组的番茄钟历史记录
formatTime()           // 格式化时间为 HH:MM
openTaskDetail()       // 打开任务详情
```

### 3. 数据来源

从 `task.pomodoroHistory` 数组中读取：
```javascript
{
  startTime: String,   // ISO格式开始时间
  endTime: String,     // ISO格式结束时间
  completed: Boolean   // 是否完成
}
```

### 5. 交互效果

- ✅ 点击日期行：展开/折叠该天的记录
- ✅ 悬停记录：左侧紫色边框高亮
- ✅ 点击记录：关闭统计弹窗，打开对应任务详情
- ✅ 空状态：显示"暂无番茄钟记录"提示

## 📊 数据统计

历史记录会自动统计：
- 每天的番茄钟数量
- 完成/放弃状态
- 实际时长（自动计算）

## 📝 注意事项

- 历史记录从 `task.pomodoroHistory` 读取
- 如果任务没有 `pomodoroHistory` 字段，会显示空状态
- 点击记录会触发 `open-task-detail` 事件，TodoView.vue 已有监听
