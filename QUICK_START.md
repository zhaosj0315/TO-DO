## 📱 安装应用

### Android 用户

1. **下载 APK**
   - 从 [GitHub Releases](https://github.com/zhaosj0315/TO-DO/releases) 下载 `TODO-App-v0.8.9.apk`

2. **安装应用**
   - 在设置中开启"允许安装未知来源应用"
   - 点击 APK 文件完成安装

3. **首次启动**
   - 打开应用
   - 注册新账号（用户名 + 密码）
   - 可选：绑定手机号或设置密保问题

### Windows 用户

1. **下载安装包**
   - 下载 `TODO App Setup 0.8.9.exe`

2. **安装应用**
   - 双击运行安装程序
   - 选择安装目录
   - 完成安装

3. **启动应用**
   - 从桌面快捷方式启动
   - 注册账号并开始使用

## 🎯 5分钟快速上手

### 1. 创建第一个任务（30秒）

1. 在顶部输入框输入任务标题
2. 选择任务类型（今天/明天/本周内等）
3. 选择分类（工作/学习/生活）
4. 选择优先级（高/中/低）
5. 点击 ✓ 按钮创建

**示例**：
```
标题：完成项目报告
类型：今天
分类：工作
优先级：高
```

### 2. 体验富媒体功能（1分钟）

#### 添加图片到任务
1. 点击任务标题进入编辑模式
2. 点击工具栏的 🖼️ **图片**按钮
3. 从相册选择一张图片
4. 系统自动插入 `![图片](local://img_xxx)` 语法
5. 点击 👁️ **预览**查看效果

#### 添加文件到任务
1. 在编辑模式下点击 📎 **文件**按钮
2. 选择文档、PDF、视频等文件（限制10MB）
3. 系统自动插入 `[📎 文件名](local://file_xxx)` 语法
4. 切换预览模式查看文件卡片

#### 预览富媒体内容
1. 点击任务中的图片 → 全屏预览
2. 点击文件卡片 → 打开预览弹窗
3. 支持PDF、视频在线预览
4. 一键下载所有文件

### 3. 使用Markdown编辑器（1分钟）

#### 基础语法体验
```markdown
# 项目计划
## 第一阶段
- [x] 需求分析
- [ ] 设计方案
- [ ] 开发实现

**重要提醒**：明天截止！

> 这是一个引用文本
```

#### 编辑和预览
1. 在任务描述中输入Markdown语法
2. 点击 👁️ **预览**按钮查看渲染效果
3. 点击 ✏️ **编辑**按钮返回编辑模式
4. 支持GitHub风格的完整语法

### 4. 管理任务（1分钟）

**完成任务**：
- 点击任务左侧的 ⭕ 圆圈标记完成

**编辑任务**：
- 点击任务标题快速编辑
- 点击任务描述区域进入详情页

**任务置顶**：
- 点击 📌 按钮将重要任务置顶

**删除任务**：
- 长按任务选择删除，移入回收站

### 5. 智能功能体验（1分钟）

#### AI智能助手
1. 点击右上角 🤖 **AI助手**按钮
2. 尝试快捷问题："今日完成情况"
3. 或输入自定义问题："我应该优先处理哪些任务？"

#### 番茄钟计时
1. 点击任务卡片上的 🍅 按钮
2. 开始25分钟专注计时
3. 全屏紫色界面，圆形进度条
```
标题：晨跑
类型：每天重复
分类：生活
优先级：中
```

### 5. AI 功能（2分钟）

**AI 问答**：
- 点击右上角 🤖 按钮
- 输入问题："今天完成了什么？"

- 点击 📷 按钮
- 拍摄或选择图片

## 🎓 进阶功能

### 任务依赖关系

**设置依赖**：
1. 打开任务详情
2. 点击"🔗 添加前置任务"
3. 选择必须先完成的任务
4. 保存

**效果**：
- 前置任务未完成时，当前任务显示 🔒 等待中
- 前置任务完成后，自动通知可以开始

---

### 任务执行日志

**添加日志**：
1. 打开任务详情
2. 点击"📝 添加日志"
3. 选择类型（开始/进展/阻碍/方案/里程碑/完成）
4. 输入内容并保存

**查看统计**：
- 任务详情中查看执行统计
- 推进次数、总耗时、阻碍统计等

---

### 番茄钟计时

**启动番茄钟**：
1. 点击任务卡片上的 🍅 按钮
2. 开始 25 分钟专注
3. 完成后自动进入休息模式

**查看统计**：
- 个人主页 → 番茄钟统计
- 查看今日/本周完成情况

---

### 数据备份

**导出数据**：
1. 个人主页 → 数据管理
2. 点击"📥 导出任务"
3. 选择保存位置

**导入数据**：
1. 点击"📤 导入任务"
2. 选择备份文件
3. 确认导入

## 💡 使用技巧

### 1. 任务优先级

- **高优先级**：紧急重要的任务（4个番茄🍅）
- **中优先级**：重要但不紧急（2个番茄🍅）
- **低优先级**：不紧急不重要（1个番茄🍅）

### 2. 任务分类

- **工作**：工作相关任务
- **学习**：学习、培训、阅读
- **生活**：日常生活、健康、娱乐

### 3. 任务类型选择

| 类型 | 适用场景 | 截止时间 |
|------|---------|---------|
| 今天 | 今天必须完成 | 今天 23:59 |
| 明天 | 明天完成 | 明天 23:59 |
| 本周内 | 本周内完成 | 本周日 23:59 |
| 指定日期 | 特定日期完成 | 指定日期时间 |
| 每天重复 | 每天都要做 | 每天 23:59 |
| 工作日重复 | 工作日做 | 每个工作日 23:59 |
| 每周重复 | 每周固定日子 | 下一个指定日 23:59 |

### 4. 重复任务最佳实践

**适合重复任务的场景**：
- 每天：晨跑、阅读、冥想
- 工作日：日报、站会、邮件检查
- 每周：周报、大扫除、聚会

**不适合重复任务的场景**：
- 一次性任务
- 不规律的任务
- 需要详细计划的项目

## 🐛 常见问题

### Q: 重复任务完成后为什么又出现了？

A: 这是正常的！重复任务完成后会在第二天/下周自动重置为待办状态，无需手动创建新任务。

### Q: 如何删除重复任务？

A: 点击任务右侧的 🗑️ 按钮删除。删除后不会再自动出现。

### Q: 任务显示 🔒 等待中是什么意思？

A: 表示这个任务依赖其他任务，需要先完成前置任务才能开始。点击任务查看详情。

### Q: 数据会丢失吗？

A: 所有数据存储在本地设备，不会上传到云端。建议定期导出备份，避免卸载应用或清除数据导致丢失。

### Q: 如何切换账号？

A: 个人主页 → 退出登录 → 登录其他账号。每个账号的数据完全隔离。
