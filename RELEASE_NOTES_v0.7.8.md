## ✨ 主要功能

### 🔗 任务依赖关系系统
- **单向依赖**: 一个任务可以"等待"另一个任务完成
- **智能通知**: 完成任务时自动通知等待的任务可以开始
- **自动清理**: 删除任务时自动清除依赖关系
- **任务跳转**: 点击等待的任务卡片可跳转到详情
- **状态显示**: 等待中🔒/被依赖🔓/无依赖✅

### 🤖 AI 任务拆分
- **智能拆解**: 输入任务标题和描述，AI自动拆分为可执行子任务
- **4种模板**: 快速拆分、详细拆分、时间优先、优先级优先
- **预估时长**: AI为每个子任务估算时长（0.5-8小时）
- **子任务预览**: Bottom Sheet展示拆分结果，支持编辑和删除
- **依赖关系**: 子任务自动依赖父任务，完成父任务后子任务才能开始

### 🎯 AI 主动式助手
- **智能提醒卡片**: 自动检测逾期和待办任务，主动提醒
- **每日智能总结**: 每日工作总结，AI建议，明日计划
- **周报/月报生成**: 自动生成结构化报告，支持导出
- **AI 对话历史**: 多对话管理，时间智能分组（今天/昨天/最近7天/更早）

### 💾 完整备份系统
- **100%数据覆盖**: 任务+日志+番茄钟+用户信息
- **一键导出/导入**: JSON格式，完整备份
- **Excel导入**: 支持中英文字段（大小写不敏感）

### 📝 任务执行日志系统
- **6种日志类型**: 开始、进展、阻碍、方案、里程碑、完成
- **无限追加日志**: 每个任务可添加多条日志记录执行过程
- **执行统计**: 追踪推进次数、总耗时、平均时长、阻碍解决情况
- **进度追踪**: 记录进度百分比
- **心情追踪**: 记录执行过程中的心情状态（顺利/一般/困难）

### 🍅 番茄钟计时器
- **25分钟专注模式**: 全屏紫色渐变计时界面
- **自动休息模式**: 5分钟短休息 / 15分钟长休息（每4个番茄钟）
- **圆形进度条**: SVG动画可视化倒计时
- **暂停/继续/放弃**: 专注过程中完全控制
- **番茄钟历史**: 追踪所有完成的专注记录（带时间戳）

### 📸 拍照OCR + AI文本增强
- **离线OCR识别**: Android MLKit离线识别（无需网络）
- **中英文混合识别**: 支持中英文混合识别
- **AI智能优化**: AI智能优化任务描述

## 🐛 Bug 修复

### 重复任务逻辑修复
- 修复 `weekday`（工作日重复）和 `weekly`（每周重复）类型任务缺少截止时间计算的问题
- 新增重复任务自动重置功能：
  - 每天重复（daily）：完成后第二天自动重置为待办
  - 工作日重复（weekday）：每个工作日自动重置
  - 每周重复（weekly）：每周指定日子自动重置
- 修复重复任务完成后不会重新出现在待办列表的问题

### Bottom Sheet 布局统一
- 修复 AddDependencyModal 为全屏宽度 Bottom Sheet
- 修复 SmartTaskSplitter 为从底部滑出布局
- 修复 WaitForSelector 为全屏 Bottom Sheet
- 优化搜索功能：支持多关键词模糊匹配
- 修复 text 字段 undefined 导致的崩溃问题

### 其他修复
- 修复AI模型配置页面重复的"添加新模型"区域
- 修复AI问答流式响应解析失败
- 修复任务详情白屏崩溃问题

## 🚀 安装说明

### Android
1. 下载 `TODO-App.apk`
2. 在Android设备上开启"允许安装未知来源应用"
3. 安装APK文件
4. 打开应用并注册新账号

### macOS
1. 下载对应芯片的 ZIP 文件
2. 解压缩
3. 将 TODO App 拖入"应用程序"文件夹
4. 首次打开时，右键点击 → 打开（绕过安全检查）

### Windows
1. 下载 `TODO App Setup 0.7.8.exe`
2. 双击运行安装程序
3. 选择安装目录并完成安装
4. 从桌面快捷方式启动应用
