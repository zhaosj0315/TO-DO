# TODO导入模板使用说明

## 📥 模板文件

**文件名**: `TODO导入模板示例.xlsx`

**用途**: 
- 作为批量导入任务的标准模板
- 包含100条示例任务，帮助用户快速了解应用功能
- 可直接导入测试应用的各项功能

## 📋 模板结构

### 必填列
| 列名 | 说明 | 示例 |
|------|------|------|
| 任务名称 | 任务标题（必填） | 朋友聚餐 |

### 可选列
| 列名 | 说明 | 可选值 | 默认值 |
|------|------|--------|--------|
| 详细描述 | 任务详细说明 | 任意文本 | 空 |
| 分类 | 任务分类 | 工作/学习/生活 | 工作 |
| 优先级 | 任务优先级 | 高/中/低 | 中 |
| 类型 | 任务周期类型 | 仅今天/每天/每周: 周一, 周三 | 仅今天 |
| 状态 | 任务状态 | 待办/已完成/已逾期 | 待办 |
| 创建时间 | 任务创建时间 | 2026/01/05 11:56 | 当前时间 |

## 🎯 使用步骤

### 1. 下载模板
从项目根目录下载 `TODO导入模板示例.xlsx`

### 2. 编辑模板（可选）
- 可以直接使用示例数据测试
- 或者修改/添加自己的任务数据
- 保持列名不变，只修改数据行

### 3. 导入数据
1. 打开TODO应用
2. 滚动到页面底部"数据管理"区域
3. 点击"📥 导入任务"按钮
4. 选择编辑好的Excel文件
5. 等待导入完成提示

### 4. 验证结果
- 查看导入提示：成功X条，失败X条
- 检查任务列表是否显示导入的任务
- 验证任务的分类、优先级、状态等信息

## ⚠️ 注意事项

1. **列名必须完全匹配**：任务名称、详细描述、分类、优先级、类型、状态、创建时间
2. **任务名称不能为空**：空行会被跳过
3. **时间格式**：支持 `2026/01/05 11:56` 或 `2026-01-05 11:56`
4. **周期格式**：每周任务格式为 `每周: 周一, 周三, 周五`
5. **编码问题**：建议使用Excel打开编辑，避免编码问题

## 🔄 导出与导入

### 导出现有任务
1. 点击"📤 导出任务"按钮
2. 自动下载Excel文件
3. 文件名格式：`TODO任务_YYYYMMDD_HHMMSS.xlsx`

### 导入任务
- 可以导入自己导出的文件
- 可以导入模板文件
- 可以导入修改后的任何符合格式的Excel文件

## 💡 使用建议

1. **首次使用**：直接导入模板文件，快速体验应用功能
2. **数据备份**：定期导出任务数据，防止数据丢失
3. **批量添加**：需要添加大量任务时，使用Excel编辑后批量导入
4. **数据迁移**：在不同设备间通过导出/导入转移数据

## 🆘 常见问题

**Q: 导入失败怎么办？**
A: 检查Excel文件列名是否正确，任务名称是否为空

**Q: 导入后任务重复了？**
A: 导入不会去重，建议导入前清空现有数据或使用新账号测试

**Q: 可以修改模板列名吗？**
A: 不可以，列名必须与模板完全一致

**Q: 支持导入多少条任务？**
A: 理论上无限制，但建议单次导入不超过1000条
