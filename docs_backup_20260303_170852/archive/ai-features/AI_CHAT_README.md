# 🤖 AI智能问答功能

## 📍 位置
右上角，演示模式按钮（💡）的左侧，紫色渐变的🤖按钮

## 🚀 快速开始
```bash
# 1. 启动应用
npm run dev

# 2. 登录并点击 🤖 按钮

# 3. 配置盘古API
模型: 盘古大模型
API地址: http://localhost:8000/api/chat

# 4. 开始提问
点击快捷问题或输入自定义问题
```

## ✨ 核心功能
- **多模型支持**: 盘古/OpenAI/本地模型
- **完整数据**: 任务/日志/番茄钟/统计
- **智能问答**: 时间/状态/分类/执行分析
- **快捷问题**: 6个常用问题一键提问
