## 🚀 使用流程

```
用户操作流程：
1. 点击🤖按钮
2. 选择模型（盘古/OpenAI/本地）
3. 配置API地址
4. 点击快捷问题或输入问题
5. 查看AI回答
6. 继续对话或关闭

数据流程：
用户问题 → 构建上下文 → 调用API → 解析响应 → 显示回答
```

## 📝 注意事项

1. **API配置** - 首次使用需要配置API地址
2. **网络要求** - 需要能访问对应的API服务
3. **数据隐私** - 数据仅在对话时发送，不会永久存储
4. **上下文限制** - 任务过多可能超出模型限制
5. **响应时间** - 根据模型和网络情况，可能需要几秒到几十秒
