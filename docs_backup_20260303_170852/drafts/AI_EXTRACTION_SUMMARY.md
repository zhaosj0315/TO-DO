## 🎯 5个AI提取场景

### 场景1: 拍照OCR识别 📷
**触发**: 点击拍照按钮 → OCR识别

**流程**: 
```
拍照 → OCR识别文字 → AITextEnhancer.enhanceText() → 填充表单
```

**提取内容**:
- ✅ 标题
- ✅ 描述
- ✅ 分类
- ✅ 优先级
- ✅ 时间（开始时间+截止时间）
- ✅ 任务类型

**示例**:
```
输入: "明天下午3点前完成项目报告"
输出: {
  title: "完成项目报告",
  description: "明天下午3点前完成",
  category: "work",
  priority: "high",
  type: "tomorrow",
  deadline: "2026-03-01 15:00",
  customDate: "2026-03-01",
  customTime: "15:00"
}
```

### 场景2: 长文本智能提取 📝
**触发**: 任务输入框输入超过20字

**流程**:
```
输入长文本 → AITaskExtractor.extractTasks() → 自动填充第一个任务
```

**提取内容**:
- ✅ 批量提取多个任务
- ✅ 每个任务包含完整信息
- ✅ 自动识别时间关系

**示例**:
```
输入: "下周一开始准备考试，周五前完成复习"
输出: [
  {
    title: "准备考试",
    startTime: "2026-03-03 09:00",
    deadline: "2026-03-07 23:59",
    ...
  }
]
```

**提取内容**: 同场景2


**提取内容**:

- ✅ 任务类型（新增）

```
  type: "this_week",

  reason: "工作相关的周报，本周五下班前完成"
```
