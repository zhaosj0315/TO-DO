# 任务卡片徽章显示规则

## 📋 4种关系徽章

任务卡片上会显示以下4种关系的数量：

### 1. 👨‍👦 父任务×1
- **条件**：`task.parentTaskId` 存在
- **含义**：这是一个子任务，由AI拆分生成
- **颜色**：紫色 `#8b5cf6`
- **示例**：`👨‍👦 父任务×1`

### 2. 👶 子任务×N
- **条件**：有子任务（`parentTaskId === task.id`）
- **含义**：这个任务已被AI拆分为N个子任务
- **颜色**：绿色 `#10b981`
- **示例**：`👶 子任务×3`

### 3. ⬆️ 依赖×N
- **条件**：`task.waitFor.length > 0`
- **含义**：这个任务依赖N个前置任务（手动添加）
- **颜色**：橙色 `#f59e0b`
- **示例**：`⬆️ 依赖×2`

### 4. ⬇️ 被依赖×N
- **条件**：其他任务的 `waitFor` 包含此任务ID
- **含义**：有N个任务依赖此任务（手动添加）
- **颜色**：蓝色 `#3b82f6`
- **示例**：`⬇️ 被依赖×5`

## 🎯 实际案例

### 案例1：简单子任务

```
任务卡片：子任务1 - 需求分析
徽章显示：👨‍👦 父任务×1
```

**解释**：这是一个子任务，来自AI拆分

---

### 案例2：已拆分的父任务

```
任务卡片：项目 - 开发新功能
徽章显示：👶 子任务×4
```

**解释**：这个任务已被拆分为4个子任务

---

### 案例3：有依赖的任务

```
任务卡片：任务B - 开始制作
徽章显示：⬆️ 依赖×1
```

**解释**：这个任务依赖1个前置任务（需要先完成任务A）

---

### 案例4：被依赖的任务

```
任务卡片：任务A - 准备材料
徽章显示：⬇️ 被依赖×3
```

**解释**：有3个任务依赖此任务（任务B、C、D都在等待任务A完成）

---

### 案例5：复杂任务（多种关系）

```
任务卡片：子任务2 - 撰写初稿
徽章显示：
  👨‍👦 父任务×1      （来自AI拆分）
  ⬆️ 依赖×1          （依赖子任务1）
  ⬇️ 被依赖×1        （子任务3依赖它）
```

**解释**：
- 这是一个子任务（父任务：写论文）
- 依赖1个前置任务（子任务1：文献调研）
- 被1个任务依赖（子任务3：修改定稿）

---

### 案例6：父任务 + 被依赖

```
任务卡片：项目 - 写论文
徽章显示：
  👶 子任务×3        （已拆分为3个子任务）
  ⬇️ 被依赖×2        （有2个其他任务依赖它）
```

**解释**：
- 这个任务已被AI拆分为3个子任务
- 有2个其他任务（非子任务）依赖它

## 📊 关系矩阵

| 任务类型 | 👨‍👦 父任务 | 👶 子任务 | ⬆️ 依赖 | ⬇️ 被依赖 |
|---------|-----------|----------|--------|----------|
| 独立任务 | ❌ | ❌ | ❌ | ❌ |
| 父任务 | ❌ | ✅ | 可选 | 可选 |
| 子任务 | ✅ | ❌ | 可选 | 可选 |
| 依赖任务 | 可选 | 可选 | ✅ | 可选 |
| 被依赖任务 | 可选 | 可选 | 可选 | ✅ |

## 🔍 徽章优先级

徽章按以下顺序显示（从左到右）：

1. 👨‍👦 父任务×1
2. 👶 子任务×N
3. ⬆️ 依赖×N
4. ⬇️ 被依赖×N
5. 💬 日志×N
6. 📊 进度×N%

---

## 💡 使用技巧

### 快速识别任务类型

- 看到 `👨‍👦` → 这是子任务，点击可查看父任务
- 看到 `👶` → 这是父任务，点击可查看所有子任务
- 看到 `⬆️` → 这个任务有前置依赖，需要等待
- 看到 `⬇️` → 这个任务很重要，其他任务在等它

### 任务规划建议

1. **项目拆解**：使用AI拆分 → 生成 `👶 子任务×N`
2. **顺序控制**：手动添加依赖 → 生成 `⬆️ 依赖×N`
3. **关键路径**：关注 `⬇️ 被依赖×N` 高的任务，优先完成
4. **进度追踪**：结合 `💬 日志×N` 和 `📊 进度×N%` 查看执行情况

## 🎨 视觉设计

### 颜色语义

| 颜色 | 关系类型 | 语义 |
|------|---------|------|
| 🟣 紫色 | 父任务 | 层级关系（向上） |
| 🟢 绿色 | 子任务 | 层级关系（向下） |
| 🟠 橙色 | 依赖 | 执行约束（等待） |
| 🔵 蓝色 | 被依赖 | 执行约束（阻塞） |

### 徽章尺寸

- 字体大小：`0.625rem`
- 内边距：`0.2rem 0.35rem`
- 圆角：`8px`
- 高度：自适应

## 📝 数据来源

```javascript
// 父任务×1
task.parentTaskId !== null

// 子任务×N
tasks.filter(t => t.parentTaskId === task.id).length

// 依赖×N
task.waitFor.length

// 被依赖×N
tasks.filter(t => t.waitFor.includes(task.id)).length
```
