## 番茄钟系统 ← 需要添加完整内容

### 核心功能
1. **25分钟专注计时器**
   - 全屏紫色渐变界面
   - 圆形SVG进度条
   - 实时倒计时显示

2. **自动休息模式**
   - 短休息：5分钟（前3个番茄钟）
   - 长休息：15分钟（每4个番茄钟）
   - 自动切换

3. **控制功能**
   - 暂停/继续
   - 放弃
   - 跳过休息
   - 继续下一个

4. **历史记录**
   - 记录所有完成的番茄钟
   - 显示开始/结束时间
   - 任务详情中展示

5. **统计功能**
   - 今日专注时长
   - 今日完成番茄钟数
   - 本周完成番茄钟数

6. **任务集成**
   - 任务卡片🍅按钮
   - 根据优先级预估番茄数
   - 完成后自动记录

### 使用流程
1. 点击任务卡片上的🍅按钮
2. 进入全屏专注界面
3. 开始25分钟倒计时
4. 完成后自动进入休息模式
5. 可选择跳过休息或继续下一个
6. 历史记录自动保存
