### 代码实际功能（v1.6.11）
通过代码审查，确认当前版本包含以下功能：

#### ✅ 已实现功能（代码中存在）
1. **每日任务摘要通知** (`scheduleDailySummaryNotification`)
   - 每天9:00自动推送
   - 统计待办/即将逾期/已逾期任务
   - 智能激励语
   - 任务变化时自动更新通知内容

2. **首次登录备份提醒** (`showBackupReminder`)
   - 新用户首次登录弹窗
   - 提醒数据备份重要性
   - 通过 Preferences 标记位控制

3. **数据说明弹窗** (`showDataInfo`)
   - 详细说明存储位置
   - 列出优缺点
   - 导出方法指南
   - 数据安全承诺

4. **导出兼容性优化** (多重降级策略)
   - Documents → ExternalStorage → Data → Browser下载
   - 修复红米手机导出失败问题

5. **页脚布局重构**
   - 3行清晰分类
   - 版本号：v1.6.11
   - 下载链接指向：`https://github.com/zhaosj0315/TO-DO/releases/tag/v1.6.11`

6. **重复任务耗时计算修复**
   - 重复任务从当天00:00计算
   - 一次性任务从创建时间计算
