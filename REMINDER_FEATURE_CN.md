# 任务提醒功能说明

## 功能概述
新增的任务提醒功能可以在指定时间响铃提醒用户，确保不会错过重要任务。

## 核心特性

### 1. 响铃提醒
- ✅ 系统默认铃声
- ✅ 持续震动（震1秒，停0.5秒，重复4次）
- ✅ 锁屏可见
- ✅ 最高优先级通知（悬浮显示）
- ✅ 红色呼吸灯

### 2. 智能提醒管理
- ✅ 创建任务时自动安排提醒
- ✅ 完成任务时自动取消提醒
- ✅ 删除任务时自动取消提醒
- ✅ 编辑提醒时间时自动重新安排
- ✅ 防重复提醒（每个任务只提醒一次）

### 3. 定时检查机制
- 每分钟检查一次是否有任务到达提醒时间
- 自动触发响铃通知
- 记录已提醒的任务，避免重复

## 使用方法

### 创建带提醒的任务
1. 在任务输入框输入任务名称
2. 勾选"🔔 启用提醒"
3. 选择提醒时间（日期+时间）
4. 点击"✓"提交

### 查看提醒状态
- 任务卡片右上角显示🔔图标表示已设置提醒
- 悬停查看具体提醒时间

### 编辑提醒
1. 点击任务卡片进入详情
2. 修改提醒时间或取消提醒
3. 保存后自动重新安排

## 技术实现

### 1. 数据结构
```javascript
task: {
  enableReminder: Boolean,  // 是否启用提醒
  reminderTime: String      // 提醒时间（ISO格式）
}
```

### 2. 核心方法
- `checkTaskReminders()`: 检查并触发到期提醒
- `scheduleTaskReminder(task)`: 安排任务提醒
- `cancelTaskReminder(taskId)`: 取消任务提醒
- `triggerReminderNotification(task)`: 触发响铃通知

### 3. 通知渠道配置
```javascript
{
  id: 'task-reminders',
  importance: 5,           // 最高优先级
  sound: 'default',        // 系统默认铃声
  vibration: true,         // 震动
  visibility: 1            // 锁屏可见
}
```

## 注意事项

### Android权限
- 首次使用需授予通知权限
- Android 13+需要额外的通知权限
- 建议在系统设置中将应用设为"不受省电限制"

### 提醒时间
- 只能设置未来时间
- 最小单位为分钟
- 建议提前5-10分钟设置提醒

### 防打扰模式
- 如果手机开启了勿扰模式，可能不会响铃
- 建议将应用加入勿扰白名单

## 测试建议

### 快速测试
1. 创建一个任务，设置1分钟后提醒
2. 等待1分钟，观察是否响铃
3. 检查通知栏是否显示提醒

### 完整测试
1. 测试创建任务时的提醒安排
2. 测试完成任务时的提醒取消
3. 测试删除任务时的提醒取消
4. 测试编辑提醒时间的重新安排
5. 测试防重复提醒机制

## 故障排查

### 不响铃？
1. 检查通知权限是否授予
2. 检查手机音量是否开启
3. 检查是否开启了勿扰模式
4. 检查应用是否被省电限制

### 重复提醒？
- 检查是否有多个相同ID的任务
- 清除应用数据重新测试

### 提醒不准时？
- Android系统可能会延迟后台任务
- 建议将应用加入省电白名单
