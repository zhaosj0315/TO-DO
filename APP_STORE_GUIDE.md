## 🌍 Google Play Store 上架流程

### 前置要求
- ✅ Google 开发者账号（$25 一次性费用）
- ✅ 签名的 APK 或 AAB 文件
- ✅ 应用图标和截图
- ✅ 隐私政策 URL

### 1. 注册 Google Play 开发者账号
1. 访问 [Google Play Console](https://play.google.com/console)
2. 使用 Google 账号登录
3. 支付 $25 注册费（一次性）
4. 填写开发者信息
5. 等待审核（通常 1-2 天）

### 2. 创建应用
1. 登录 Play Console
2. 点击 **创建应用**
3. 填写基本信息：
   - 应用名称：TODO App
   - 默认语言：中文（简体）或英语
   - 应用类型：应用
   - 免费/付费：免费

### 3. 准备应用资源

#### 必需资源
- **应用图标**: 512x512 PNG（32位，带透明度）
- **功能图片**: 1024x500 JPG/PNG
- **手机截图**: 至少 2 张，最多 8 张
  - 尺寸：320-3840 像素
  - 推荐：1080x1920 或 1080x2340
- **7 英寸平板截图**: 至少 2 张（可选）
- **10 英寸平板截图**: 至少 2 张（可选）

#### 应用描述
- **简短描述**: 最多 80 字符
  ```
  离线待办事项管理，支持番茄钟、多分类、优先级管理
  ```
- **完整描述**: 最多 4000 字符
  ```
  TODO App 是一款功能强大的离线待办事项管理应用。

  核心功能：
  • 完全离线运行，数据安全可靠
  • 7种任务类型：今天、明天、本周、指定日期、每天重复、工作日重复、每周重复
  • 番茄钟激励系统，根据优先级自动计算番茄数
  • 任务分类：工作、学习、生活
  • 优先级管理：高、中、低
  • 截止时间提醒，颜色分级显示
  • 高级筛选：按状态、分类、优先级、日期、关键字搜索
  • 数据导入导出，支持 Excel 格式
  • 回收站功能，误删可恢复
  • 个人主页统计，查看任务完成情况

  特色亮点：
  ✓ 无需网络，完全离线使用
  ✓ 数据存储在本地，隐私安全
  ✓ 界面简洁美观，操作流畅
  ✓ 支持多种任务周期
  ✓ 番茄钟成就系统，激励完成任务

  适用人群：
  • 学生：管理学习任务和作业
  • 职场人士：规划工作事项
  • 自由职业者：安排日程计划
  • 所有需要高效时间管理的人

  立即下载，开启高效生活！
  ```

### 4. 设置内容分级
1. 填写内容分级问卷
2. 对于 TODO App，选择：
   - 暴力：无
   - 性内容：无
   - 语言：无
   - 受管制物品：无
   - 其他：无
3. 获得分级：所有人（3+）

### 5. 设置目标受众和内容
- **目标年龄组**: 所有年龄段
- **是否包含广告**: 否
- **应用访问权限**: 说明需要的权限（存储、通知等）

### 6. 隐私政策
**必需！** 需要提供隐私政策 URL。

创建简单的隐私政策页面，包含：
- 收集的数据类型（本地存储）
- 数据使用方式
- 数据安全措施
- 联系方式

示例：
```
隐私政策

TODO App 尊重并保护用户隐私。

数据收集：
本应用完全离线运行，不收集任何用户数据。所有任务数据仅存储在您的设备本地。

数据使用：
应用不会上传、分享或出售您的任何个人信息。

权限说明：
• 存储权限：用于保存任务数据到本地
• 通知权限：用于任务提醒（可选）

联系我们：
如有疑问，请联系：your-email@example.com

更新日期：2026-02-19
```

### 7. 上传 APK/AAB

#### 生成签名的 Release APK
```bash
cd android
./gradlew assembleRelease
```

APK 位置：`android/app/build/outputs/apk/release/app-release.apk`

#### 或生成 AAB（推荐）
```bash
cd android
./gradlew bundleRelease
```

AAB 位置：`android/app/build/outputs/bundle/release/app-release.aab`

**注意**: Google Play 现在推荐使用 AAB 格式。

#### 上传到 Play Console
1. 进入 **生产** > **版本**
2. 点击 **创建新版本**
3. 上传 APK 或 AAB 文件
4. 填写版本说明：
   ```
   v1.4.0 首次发布
   
   • 完整的离线待办事项管理功能
   • 番茄钟激励系统
   • 7种任务类型支持
   • 高级筛选和搜索
   • 数据导入导出
   ```
5. 点击 **保存** 和 **审核版本**

### 8. 提交审核
1. 完成所有必填项
2. 点击 **提交审核**
3. 等待审核结果（通常 1-7 天）

### 9. 审核通过后
- 应用会在 Google Play 上线
- 用户可以搜索并下载
- 可以在 Play Console 查看下载量、评分等数据

## 🇨🇳 中国应用商店上架流程

### 1. 华为应用市场

#### 注册开发者账号
1. 访问 [华为开发者联盟](https://developer.huawei.com)
2. 注册并实名认证
3. 个人开发者：免费

1. 登录开发者后台
2. 创建应用
3. 填写应用信息：
   - 应用名称
   - 应用简介
   - 应用详情
   - 应用分类：效率办公
   - 应用标签：待办事项、时间管理
4. 上传资源：
   - 应用图标：512x512
   - 应用截图：至少 3 张
   - 宣传图：可选
5. 上传 APK
6. 填写版本信息
7. 提交审核

#### 审核时间
- 通常 1-3 个工作日

## ⚠️ 常见审核问题

### 1. 权限问题
- 只申请必要的权限
- 在隐私政策中说明权限用途

### 2. 内容合规
- 不包含违法违规内容
- 不包含政治敏感内容
- 不包含色情暴力内容

### 3. 功能完整性
- 应用功能完整可用
- 没有明显 Bug
- 界面友好

### 4. 资质要求
- 某些类别需要特殊资质（如金融、医疗）
- TODO App 属于工具类，通常无需特殊资质
