## 📱 安装与启动

### Android 设备安装
1. **下载**: 获取 `TODO-App.apk` 安装包。
2. **授权**: 在系统设置中开启"允许安装未知来源应用"。
3. **安装**: 点击 APK 文件完成安装。
4. **注册**: 首次使用需注册，支持设置**密保问题**或**绑定手机号**（可选）。

### Windows 桌面端安装
1. **下载**: 运行 `TODO App Setup 1.6.0.exe`。
2. **启动**: 安装后从桌面快捷方式打开，体验与移动端同步的离线管理。

### Mac 桌面端安装
1. **下载**: 获取 `TODO-App-1.6.0-mac.dmg` 安装包。
2. **安装**: 双击 DMG 文件，将应用拖入 Applications 文件夹。
3. **启动**: 从启动台或应用程序文件夹打开 TODO App。

## 🎯 功能使用指南

### 1. 账户与安全
- **动态头像**: 登录后，系统根据用户名首字母生成专属渐变头像。
- **手机绑定**: 在【个人主页】可绑定手机号，绑定后支持通过手机号+验证码找回账号或登录。
- **修改密码**: 登录状态下，可在个人主页进行密码变更。
- **数据隔离**: 多用户共用设备时，任务数据按账号严格隔离，互不干扰。

### 2. 任务全能管理
- **多样化类型**: 支持 7 种任务模式：今天、明天、本周内、指定日期、每天重复、工作日重复、每周重复。
- **全字段编辑**: 点击任务标题可修改名称、详细描述、分类、优先级、类型及具体的日期时间。
- **截止时间预警**: 
  - ⏰ 自动计算剩余时间。
  - 颜色分级：蓝色（正常）、橙色（警告：<3h）、红色（紧急：<1h）。
  - 紧急任务会伴随脉冲动画提醒。

### 3. 智能筛选与搜索 (v1.6.0优化)
- **统计栏筛选** (v1.6.0视觉升级): 
  - 第一行统计栏包含：[全部] [已完成] [待办] [已逾期] [🔍筛选] [▼展开/收起]
  - 核心指标（全部、已逾期）宽度+20%，更加醒目
  - 色彩语义化：已完成(淡绿)、待办(淡蓝)、已逾期(淡红)背景
  - 点击任意统计项可快速筛选对应状态的任务
  - **待办筛选** (v1.5.9修复): 现在只显示真正 pending 状态的任务，不再包含已逾期任务
- **高级筛选** (v1.6.0紧凑化): 
  - 点击统计栏右侧的 **🔍筛选按钮** 打开高级筛选弹窗
  - 弹窗采用紧凑设计，模块间距压缩44%，信息密度提升
  - 支持按日期范围、分类（工作/学习/生活）、优先级（高/中/低）组合筛选
  - 支持关键字搜索（模糊匹配任务名称和描述）
- **搜索框**: 第二行独立搜索框，可快速搜索任务
- **刷新按钮** (v1.6.0优化): 点击右上角紫色刷新按钮，重置所有筛选条件到初始状态并重新加载数据

### 4. 番茄钟激励系统 🍅
- **赚取番茄**: 
  - 完成高优先级任务可获得 **4个** 番茄。
  - 中优先级获得 **2个**，低优先级 **1个**。
- **等级勋章**: 累计获得的"净番茄数"将决定您的等级：
  - 🌱 番茄新手 (0+) -> 🌟 番茄新星 (50+) -> ⭐ 番茄达人 (150+) -> 🏆 番茄专家 (300+) -> 👑 番茄大师 (500+)。
- **成就统计**: 在个人主页点击"番茄钟统计"查看近7天完成趋势、连续打卡天数及分类占比。

### 5. 智能提醒与下拉刷新
- **系统通知**: 任务即将逾期（1小时前）或已逾期时，系统会弹出通知（包含幽默提醒话术）。
- **手势操作**: 在任务列表页面下拉即可触发"下拉刷新"，更新逾期状态。

## 📊 数据管理

### 1. 数据导出与备份
1. 进入【个人主页】->【数据管理】。
2. 点击 **📥 导出任务**，系统将生成 Excel 文件并保存至手机"文档"目录。
3. 文件命名格式：`TODO任务_用户名_日期.xlsx`。

### 2. 批量导入任务
1. 点击 **📋 下载模板**，获取官方 Excel 格式。
2. 填入任务信息后，点击 **📤 导入任务** 选择该文件。
3. 系统将自动解析并批量创建任务，支持识别分类、优先级及周期设置。

## 🐛 常见问题 FAQ

**Q: 为什么任务会变红？**  
A: 当任务超过截止时间且未完成时，会标记为"已逾期"状态，并扣除相应的番茄分。

**Q: 换手机了数据能同步吗？**  
A: 不支持自动同步。请在旧手机导出 Excel，在新手机导入即可完成迁移。

**Q: 绑定的手机号收不到短信？**  
A: 本应用为离线模拟。验证码会通过系统通知栏弹出，请注意查看通知中心。

**Q: 点击"待办"为什么显示的任务数量和统计不一致？(v1.5.9已修复)**  
A: 此问题已在v1.5.9版本修复。现在点击"待办"只会显示真正pending状态的任务，不再包含已逾期的任务。

**Q: 筛选按钮在哪里？(v1.5.8更新)**  
A: 筛选按钮（🔍）已从搜索栏移至第一行统计栏，位于"已逾期"和"展开/收起箭头"之间，点击可打开高级筛选弹窗。
