### 2.2 Dead Code示例

#### 高风险（确认未使用）
```javascript
// src/stores/offlineTaskStore.js:370
clearAllDeletedTasks()  // 0引用

// src/stores/taskStore.js:247
updateTaskStatus()  // 0引用（废弃的Store）

// src/stores/userStore.js:39
loadUser()  // 0引用
```

#### 中风险（需人工确认）
```javascript
// src/utils/autoBackup.js:15
getTodayDate()  // 可能被动态调用

// src/composables/useTextSelection.js:155
closeMenu()  // 可能在模板中使用
```

### 2.3 安全策略

#### ⚠️ 不要删除的情况
1. **动态调用**: `this[funcName]()`, `obj[key]()`
2. **字符串引用**: `'functionName'` 作为参数
3. **模板引用**: Vue模板中的 `@click="funcName"`
4. **导出函数**: `export { funcName }`
5. **事件处理器**: `addEventListener('click', funcName)`
6. **生命周期钩子**: Vue/React生命周期方法

#### ✅ 删除步骤
1. 人工确认函数确实未使用
2. 使用Git创建备份分支: `git checkout -b cleanup-dead-code`
3. 删除函数代码
4. 运行测试: `npm test`
5. 手动测试关键功能
6. Code Review
7. 合并到主分支
