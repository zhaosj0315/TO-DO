# 日历任务下钻功能实现

**实现日期**: 2026-03-12  
**功能**: 日历中的任务支持点击跳转到任务详情页面

## 📋 实现内容

### 1. CalendarView.vue 修改

#### ✅ 添加 openTask 事件
```javascript
// 定义 emit
const emit = defineEmits(['close', 'openTask'])

// 打开任务详情
function openTaskDetail(task) {
  closeDayTasks()
  emit('close')
  emit('openTask', task)
}
```

**逻辑说明**:
1. 关闭日期任务列表弹窗
2. 关闭日历主弹窗
3. 触发 `openTask` 事件，传递任务对象给父组件

### 2. TodoView.vue 修改

#### ✅ 监听 openTask 事件
```vue
<CalendarView
  v-if="showCalendar"
  @close="showCalendar = false"
  @openTask="openTaskFromCalendar"
/>
```

#### ✅ 添加处理函数
```javascript
// 方法：从日历打开任务详情
const openTaskFromCalendar = (task) => {
  openTaskDetail(task)
}
```

**复用现有逻辑**: 直接调用已有的 `openTaskDetail` 函数，无需重复实现

## 🎯 用户交互流程

```
1. 用户点击"📅 日历"按钮
   ↓
2. 日历弹窗从底部滑出
   ↓
3. 用户点击某个日期
   ↓
4. 显示该日期的任务列表（底部弹窗）
   ↓
5. 用户点击某个任务
   ↓
6. 关闭日期任务列表
   ↓
7. 关闭日历主弹窗
   ↓
8. 打开任务详情弹窗（TaskDetailModal）
```

## ✅ 功能特性

### 支持的操作
- ✅ 点击待办任务 → 打开任务详情
- ✅ 点击已完成任务 → 打开任务详情（只读）
- ✅ 任务详情中可以编辑、删除、添加日志等
- ✅ 关闭任务详情后返回首页（不返回日历）

### 交互优化
- 自动关闭所有日历相关弹窗
- 无缝切换到任务详情
- 保持任务数据同步

## 📱 测试场景

1. ✅ 打开日历 → 选择日期 → 点击待办任务 → 查看详情

3. ✅ 任务详情中编辑任务 → 保存 → 返回首页

5. ✅ 任务详情中添加日志 → 保存 → 查看日志列表
